## Introduction: The CEO's Value Creation Mandate

## Practice Exercises

### Exercise 1: Value Driver Analysis

For a company you know well:
1. Build a value driver tree
2. Identify the 3-5 highest leverage drivers
3. Assess current performance on each
4. Develop improvement initiatives

### Exercise 2: Portfolio Assessment

For a multi-business company:
1. Map businesses on growth/ROIC matrix
2. Assess parenting advantage for each
3. Recommend hold/invest/divest for each
4. Develop capital allocation plan

### Exercise 3: Synergy Planning

For a proposed acquisition:
1. Identify all potential synergies
2. Estimate value and timing for each
3. Discount for execution risk
4. Compare to likely acquisition premium
