## Introduction: The Compensation Challenge

Executive compensation is among the most scrutinized aspects of corporate governance. Properly designed, it aligns management interests with shareholders and stakeholders. Poorly designed, it can destroy value, create perverse incentives, and damage corporate reputation.

## Long-Term Incentive Vehicles

### Equity Compensation Types

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    EQUITY COMPENSATION VEHICLES                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   STOCK OPTIONS                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Right to buy shares at grant price (exercise price)          │  │
│   │ • Value = Stock price - Exercise price (if positive)           │  │
│   │ • Typically vest over 3-4 years                                │  │
│   │ • Expire after 7-10 years                                      │  │
│   │                                                                 │  │
│   │ Pros: Upside leverage, no cost if stock falls                  │  │
│   │ Cons: Underwater options have no retention value               │  │
│   │       Incentivize risk-taking, dilutive                        │  │
│   │                                                                 │  │
│   │ Example: 10,000 options at $100 exercise price                 │  │
│   │          If stock goes to $150 → Value = $500,000              │  │
│   │          If stock falls to $80 → Value = $0                    │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   RESTRICTED STOCK UNITS (RSUs)                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Promise to deliver shares after vesting                      │  │
│   │ • Value = Current stock price                                  │  │
│   │ • Typically time-based vesting (3-4 years)                     │  │
│   │ • Value retained even if stock declines                        │  │
│   │                                                                 │  │
│   │ Pros: Retention value, simpler valuation                       │  │
│   │ Cons: Less leverage, value even for poor performance           │  │
│   │                                                                 │  │
│   │ Example: 5,000 RSUs granted at $100/share                      │  │
│   │          If stock goes to $150 → Value = $750,000              │  │
│   │          If stock falls to $80 → Value = $400,000              │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   PERFORMANCE SHARE UNITS (PSUs)                                        │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Shares earned based on performance metrics                   │  │
│   │ • 3-year performance period typical                            │  │
│   │ • Payout range: 0% to 200% of target                           │  │
│   │ • Metrics: TSR, EPS growth, ROIC, revenue                      │  │
│   │                                                                 │  │
│   │ Pros: Strong pay-for-performance, long-term focus              │  │
│   │ Cons: Complex, goal-setting challenges                         │  │
│   │                                                                 │  │
│   │ Example: 5,000 target PSUs                                     │  │
│   │          Top quartile TSR → 200% = 10,000 shares               │  │
│   │          Median TSR → 100% = 5,000 shares                      │  │
│   │          Bottom quartile → 0% = 0 shares                       │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### LTI Vehicle Comparison

| Feature | Stock Options | RSUs | PSUs |
|---------|--------------|------|------|
| **Value when granted** | $0 (no intrinsic) | Market price | Target value |
| **Performance link** | Stock price only | Stock price only | Metrics + stock |
| **Downside protection** | None | Retains value | None (0-200%) |
| **Upside leverage** | High | Low | Medium |
| **Dilution** | High | Medium | Medium |
| **Accounting** | Black-Scholes | Grant price | Fair value |
| **Popularity trend** | Declining | Stable | Increasing |

## Compensation Governance

### Role of the Compensation Committee

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    COMPENSATION COMMITTEE RESPONSIBILITIES              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   CEO COMPENSATION                                                      │
│   ├── Establish CEO pay philosophy and strategy                         │
│   ├── Set CEO base salary                                               │
│   ├── Determine annual incentive targets and payouts                    │
│   ├── Grant long-term incentive awards                                  │
│   ├── Evaluate CEO performance                                          │
│   └── Review/approve employment agreements                              │
│                                                                         │
│   EXECUTIVE TEAM                                                        │
│   ├── Approve pay for other NEOs (Named Executive Officers)             │
│   ├── Review succession planning for key executives                     │
│   ├── Oversee executive development programs                            │
│   └── Approve executive agreements and severance                        │
│                                                                         │
│   PLAN DESIGN                                                           │
│   ├── Review and approve incentive plan designs                         │
│   ├── Select performance metrics and targets                            │
│   ├── Establish equity grant guidelines                                 │
│   ├── Approve clawback and ownership policies                           │
│   └── Monitor plan effectiveness                                        │
│                                                                         │
│   OVERSIGHT                                                             │
│   ├── Engage independent compensation consultant                        │
│   ├── Assess compensation risk                                          │
│   ├── Monitor regulatory developments                                   │
│   ├── Prepare CD&A for proxy statement                                  │
│   └── Respond to say-on-pay results and shareholder feedback            │
│                                                                         │
│   GOVERNANCE                                                            │
│   ├── Maintain committee charter                                        │

### Responding to Low Say-on-Pay Support

```
┌─────────────────────────────────────────────────────────────────────────┐

│   IF VOTE FAILS OR SUPPORT <70%:                                        │
│                                                                         │
│   PHASE 1: UNDERSTAND (0-30 Days)                                       │

│   └── Assess media and stakeholder reaction                             │

│   │   • Holding period requirements                                     │

## Executive Pay Controversies

### Common Issues and Best Practice Responses

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    COMPENSATION BEST PRACTICES                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   PAY FOR FAILURE                                                       │
│   Issue: Large payouts despite poor performance                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ Best Practices:                                                 │  │
│   │ • Performance-based severance                                   │  │
│   │ • Forfeiture of unvested equity on termination for cause        │  │
│   │ • No "walk-away" payments for voluntary departure               │  │
│   │ • Clawback policies                                             │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   EXCESSIVE CHANGE-IN-CONTROL                                           │
│   Issue: "Golden parachutes" rewarding executives in M&A                │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ Best Practices:                                                 │  │
│   │ • Double trigger (CIC + termination required)                   │  │
│   │ • No excise tax gross-ups                                       │  │
│   │ • Reasonable multiples (2-3x, not 5x)                           │  │
│   │ • Cap at 2.99x to avoid Section 280G penalties                  │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   PROBLEMATIC PRACTICES                                                 │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ Avoid:                                                          │  │
│   │ ✗ Stock option repricing                                        │  │
│   │ ✗ Backdating grants                                             │  │
│   │ ✗ Reload options                                                │  │
│   │ ✗ Tax gross-ups (except for relocation)                         │  │
│   │ ✗ Excessive perquisites                                         │  │
│   │ ✗ Supplemental pensions without justification                   │  │
│   │ ✗ Guaranteed bonuses                                            │  │
│   │ ✗ Single-trigger change-in-control vesting                      │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   ALIGNMENT PRACTICES                                                   │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ Adopt:                                                          │  │
│   │ ✓ Meaningful stock ownership requirements                       │  │
│   │ ✓ Holding requirements post-vesting                             │  │
│   │ ✓ Anti-hedging/anti-pledging policies                           │  │
│   │ ✓ Robust clawback policies                                      │  │
│   │ ✓ Rigorous performance goals                                    │  │
│   │ ✓ Multi-year performance periods                                │  │
│   │ ✓ Caps on incentive payouts                                     │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
