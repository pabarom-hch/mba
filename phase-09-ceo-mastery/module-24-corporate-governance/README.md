## Module Overview

Corporate governance defines the system of rules, practices, and processes by which companies are directed and controlled. As a CEO or board member, understanding governance ensures you balance stakeholder interests, maintain accountability, and create sustainable long-term value. This module covers the theoretical foundations, practical frameworks, and emerging trends that shape modern corporate governance.

## Theoretical Foundations

### Agency Theory

Agency theory addresses the fundamental problem arising when ownership and control are separated in modern corporations.

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         AGENCY RELATIONSHIP                             │
├─────────────────────────────────────────────────────────────────────────┤

│   │ • Seek max returns  │ ──────────►  │ • Have information  │         │
│   │ • Bear residual risk│   Authority  │ • May have own goals│         │

### Agency Costs

**Types of Agency Costs:**
1. **Monitoring costs** - Expenses incurred by principals to observe agent behavior
2. **Bonding costs** - Expenses incurred by agents to demonstrate trustworthiness
3. **Residual loss** - Value destruction from imperfect alignment despite controls

### Shareholder vs. Stakeholder Primacy

| Dimension | Shareholder Primacy | Stakeholder Model |
|-----------|--------------------|--------------------|
| **Primary duty** | Maximize shareholder value | Balance all stakeholder interests |
| **Legal basis** | Fiduciary duty to owners | Corporate purpose statutes |
| **Time horizon** | Quarterly/annual returns | Long-term sustainability |
| **Risk management** | Financial risk focus | Enterprise risk including ESG |
| **Success metrics** | Stock price, EPS, ROE | Triple bottom line |
| **Critics say** | Short-termism, externalities | Accountability diffusion |

## Key Governance Principles

```
┌─────────────────────────────────────────────────────────────────────────┐
│               OECD PRINCIPLES OF CORPORATE GOVERNANCE                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   1. EFFECTIVE GOVERNANCE FRAMEWORK                                     │
│   ├── Promote transparent markets                                       │
│   ├── Consistent with rule of law                                       │
│   └── Clear division of responsibilities                                │
│                                                                         │
│   2. SHAREHOLDER RIGHTS                                                 │
│   ├── Secure ownership registration                                     │
│   ├── Participate in fundamental decisions                              │
│   └── Information and voting rights                                     │
│                                                                         │
│   3. INSTITUTIONAL INVESTORS                                            │
│   ├── Disclose voting policies                                          │

## Real-World Governance Failures

Understanding past failures illuminates why governance matters:

| Company | Year | Failure | Consequence |
|---------|------|---------|-------------|
| Enron | 2001 | Board captured, audit failure | $74B value destroyed |
| WorldCom | 2002 | Accounting fraud, weak board | $180B loss, bankruptcy |
| Lehman Brothers | 2008 | Risk oversight failure | Triggered financial crisis |
| Volkswagen | 2015 | Emissions fraud, dual board failure | $30B+ costs |
| Wells Fargo | 2016 | Incentive misalignment | $3B+ fines, reputation damage |
| Theranos | 2018 | Board oversight, founder control | Criminal convictions |
| Wirecard | 2020 | Audit failure, regulatory capture | $2B missing, bankruptcy |
