### Advisor Input

The board's advisors provide their assessments:

**Investment Bank (Defense Advisor):**
- Crestview has a strong track record (60% success rate)
- Sterling is vulnerable on performance grounds
- Sum-of-parts analysis suggests modest discount
- Recommend proactive engagement to control narrative
- Prepare for proxy fight but seek settlement

**Legal Counsel:**
- No poison pill in place; could adopt if needed
- Advance notice bylaws would require 90-day notice for nominations
- Universal proxy rules will apply; harder to defend
- Recommend understanding shareholder base sentiment

**Proxy Solicitor:**
- Surveyed top 15 shareholders (holding 52%)
- 8 expressed sympathy for some Crestview arguments
- 5 expressed concern about Consumer Products
- 3 expressed concern about board tenure

## Decision Point

The board must now decide how to respond. Management has prepared several options:

### Option A: Full Resistance

- Reject all Crestview demands
- Adopt poison pill as defensive measure
- Prepare for proxy fight
- Defend current strategy and leadership

### Option B: Selective Engagement

- Engage with Crestview in dialogue
- Consider some proposals on merits
- Make voluntary governance improvements
- Develop enhanced shareholder value plan

### Option C: Settlement

- Negotiate with Crestview toward settlement
- Accept 1-2 board seats (potentially mutual candidates)
- Commit to strategic review of Consumer Products
- Announce governance and compensation changes

---

## Discussion Questions

### Strategic Questions

**Question 1:** Evaluate Crestview's core thesis. Is the company actually underperforming, or is this just a reflection of market cycles? What evidence would you use to make this assessment?

**Question 2:** Should Sterling divest Consumer Products? What are the pros and cons? How should the board evaluate this decision independently of activist pressure?

**Question 3:** Crestview proposes increasing leverage from 0.3x to 2.0x net debt/EBITDA. What are the strategic implications of this change? Is this appropriate for an industrial company?

### Governance Questions

**Question 4:** The CEO has been in role for 12 years with declining relative performance. How should the board evaluate his leadership? What process should be followed?

**Question 5:** Should the board separate the CEO and Chair roles? What factors should inform this decision?

**Question 6:** Three directors have tenure over 12 years. Is this a problem? How should the board approach refreshment?

### Tactical Questions

**Question 7:** Which response option (A, B, or C) would you recommend? Why?

**Question 8:** If you recommend settlement, what specific terms would you propose? What would you accept vs. reject from Crestview's demands?

**Question 9:** How should the board communicate with shareholders throughout this process?

**Question 10:** Crestview has proposed three director candidates. How should the board evaluate them? Under what circumstances would you accept their nominees vs. insisting on mutual candidates?

## Lessons Learned

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    CASE LESSONS                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   1. EVALUATE DEMANDS ON MERITS                                         │
│      • Activist proposals should be judged by content, not source       │
│      • Some demands may be value-creating regardless of motivation      │
│      • Board should have independent analysis of all options            │
│                                                                         │
│   2. PROACTIVE ENGAGEMENT WORKS                                         │
│      • Early, constructive dialogue prevents escalation                 │
│      • Shareholder relationships matter before crisis                   │
│      • Transparency about challenges builds credibility                 │
│                                                                         │
│   3. SETTLEMENT OFTEN MAKES SENSE                                       │
│      • Proxy fights are expensive and distracting                       │
│      • Outcomes are uncertain                                           │
│      • Well-structured settlement can benefit all parties               │
│                                                                         │
│   4. GOVERNANCE MATTERS                                                 │
│      • Board composition and tenure affect vulnerability                │
│      • Pay-for-performance alignment prevents criticism                 │
│      • Proactive refreshment is better than forced change               │
│                                                                         │
│   5. PERFORMANCE IS THE BEST DEFENSE                                    │
│      • Underperformance invites activism                                │
│      • Credible strategic plan reduces vulnerability                    │
│      • Execution and accountability are essential                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
