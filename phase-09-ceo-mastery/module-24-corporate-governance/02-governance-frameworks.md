## Sarbanes-Oxley Act (SOX)

### Overview and Context

The Sarbanes-Oxley Act of 2002 was enacted in response to major corporate scandals (Enron, WorldCom) and fundamentally transformed corporate governance in the United States.

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    SARBANES-OXLEY KEY PROVISIONS                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   TITLE I: PUBLIC COMPANY ACCOUNTING OVERSIGHT BOARD (PCAOB)           │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Created PCAOB to oversee public company auditors              │  │
│   │ • Auditor registration and inspection requirements              │  │
│   │ • Auditing, quality control, and independence standards         │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   TITLE II: AUDITOR INDEPENDENCE                                        │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Prohibited non-audit services by auditors                     │  │
│   │ • Mandatory audit partner rotation (5 years)                    │  │
│   │ • Audit committee pre-approval of all services                  │  │
│   │ • Cooling-off period for auditor employment                     │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   TITLE III: CORPORATE RESPONSIBILITY                                   │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Audit committee independence (all members)                    │  │
│   │ • Audit committee financial expert requirement                  │  │
│   │ • CEO/CFO certification of financial statements                 │  │
│   │ • Prohibition on personal loans to executives                   │  │
│   │ • Forfeiture of bonuses if restatement due to misconduct        │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   TITLE IV: ENHANCED FINANCIAL DISCLOSURES                              │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ § 302: CEO/CFO must certify financial statements                │  │
│   │ § 404: Management assessment of internal controls               │  │
│   │ § 404: Auditor attestation of internal controls                 │  │
│   │ • Off-balance sheet disclosure                                  │  │
│   │ • Pro forma financial presentation rules                        │  │
│   │ • Insider trading blackout periods                              │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   TITLE VIII-XI: PENALTIES AND ENFORCEMENT                              │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Criminal penalties for securities fraud (up to 25 years)      │  │
│   │ • Destruction of audit records (up to 20 years)                 │  │
│   │ • Whistleblower protections                                     │  │
│   │ • Extended statute of limitations                               │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Comply or Explain vs. Rules-Based

| Aspect | Comply or Explain | Rules-Based |
|--------|-------------------|-------------|
| **Flexibility** | High - allows deviation with explanation | Low - must comply |
| **Enforcement** | Market forces, disclosure | Regulatory penalties |
| **Innovation** | Enables governance innovation | Standardized approach |
| **Accountability** | Through explanation quality | Through compliance |
| **Examples** | UK, Australia, Germany | US (SOX, listing rules) |
| **Risk** | Boilerplate explanations | Box-ticking compliance |

### Exercise 1: SOX Compliance Assessment

**Scenario:** You are the newly appointed CFO of a mid-size public company. Evaluate the following situations and identify SOX compliance concerns:

1. The audit committee includes the CEO's brother-in-law who is a retired CPA
2. The external auditor also provides tax consulting services worth $2M annually
3. The company has no formal process to document or test internal controls
4. The CEO wants to delay reporting a material acquisition until next quarter

### Exercise 2: Internal Control Framework

Design a basic internal control framework for the revenue cycle:

**Required elements:**
- Control environment considerations
- Key risks to assess
- Control activities (preventive and detective)
- Information and communication requirements
- Monitoring activities

You are reviewing your company's Corporate Governance Guidelines. Identify what's missing:

*Current document covers:*
- Board size (5-15 members)
- Director term (1 year)
- Annual meeting frequency
- Committee structure

**Questions:**
1. What essential elements are missing?
2. What NYSE requirements might be violated?
