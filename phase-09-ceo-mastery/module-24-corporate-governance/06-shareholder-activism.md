By the end of this lesson, you will be able to:

5. **Navigate** proxy contests and settlement negotiations

### Typical Activist Playbook

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    ACTIVIST CAMPAIGN TIMELINE                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   PHASE 1: TARGET IDENTIFICATION (Months -6 to -3)                      │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Screen for undervalued companies                              │  │
│   │ • Analyze fundamentals and governance                           │  │
│   │ • Identify catalysts for value creation                         │  │
│   │ • Assess vulnerability and defensibility                        │  │
│   │ • Evaluate shareholder base and likely support                  │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              │                                          │
│                              ▼                                          │
│   PHASE 2: STAKE BUILDING (Months -3 to 0)                              │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Accumulate shares (often through derivatives)                 │  │
│   │ • Stay below 5% to avoid Schedule 13D filing                    │  │
│   │ • Continue due diligence                                        │  │
│   │ • Prepare campaign materials                                    │  │
│   │ • Retain advisors (legal, PR, proxy solicitor)                  │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              │                                          │
│                              ▼                                          │
│   PHASE 3: PRIVATE ENGAGEMENT (Months 0 to +3)                          │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • File Schedule 13D (if >5%)                                    │  │
│   │ • Send private letter to board                                  │  │
│   │ • Request meetings with management and board                    │  │
│   │ • Present investment thesis and demands                         │  │
│   │ • Explore settlement options                                    │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              │                                          │
│                              ▼                                          │
│   PHASE 4: PUBLIC CAMPAIGN (If no settlement)                           │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Release public letter / presentation                          │  │
│   │ • Media campaign (press, interviews)                            │  │
│   │ • Investor outreach                                             │  │
│   │ • Nominate director candidates (proxy fight)                    │  │
│   │ • File shareholder proposals                                    │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              │                                          │
│                              ▼                                          │
│   PHASE 5: PROXY CONTEST (If no settlement)                             │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • File proxy materials with SEC                                 │  │
│   │ • Universal proxy card (new rules)                              │  │
│   │ • Solicit shareholder votes                                     │  │
│   │ • Campaign through annual meeting                               │  │
│   │ • Shareholder vote determines outcome                           │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                              │                                          │
│                              ▼                                          │
│   PHASE 6: POST-CAMPAIGN                                                │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Implement changes (if successful)                             │  │
│   │ • Monitor progress                                              │  │
│   │ • Exit position when value realized                             │  │
│   │ • Or continue engagement for next annual meeting                │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Activist Target Profile

Companies most likely to be targeted share these characteristics:

| Factor | Target Profile |
|--------|---------------|
| **Size** | $1B-$50B market cap (sweet spot) |
| **Performance** | TSR below peers for 3+ years |
| **Valuation** | Trading below intrinsic value |
| **Margins** | Below industry average |
| **Balance sheet** | Net cash or under-leveraged |
| **Structure** | Conglomerate or non-core assets |
| **Governance** | Weak or entrenched board |
| **Ownership** | High institutional, low insider |

```
│   FOR ACTIVIST:                     FOR COMPANY:                        │

│   │• Credible thesis       │        │• Strong performance     │         │
│   │• Quality candidates    │        │• Credible plan          │         │
│   │• ISS/Glass Lewis       │        │• Unified board          │         │
│   │  support               │        │• Shareholder support    │         │
│   │• Institutional backing │        │• Effective communication│         │
│   │• Effective messaging   │        │• Discredit activist     │         │
│   └────────────────────────┘        │  (carefully)            │         │
│                                     └────────────────────────┘         │
│                                                                         │
│   TYPICAL OUTCOMES:                                                     │
│   • ~15% go to vote (most settle before)                                │
│   • Activists win ~40% of votes that occur                              │
│   • Settlements often include 1-2 board seats                           │
│   • Full slate victories by activists are rare                          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Settlement Considerations

```
│                    SETTLEMENT DECISION FRAMEWORK                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   REASONS TO SETTLE:                                                    │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Activist has some valid points                                │  │
│   │ • Board seat(s) won't change board dynamics materially          │  │
│   │ • Avoid costly, distracting proxy fight                         │  │
│   │ • Shareholder base may support activist                         │  │
│   │ • Proxy advisors likely to support activist                     │  │
│   │ • Uncertainty of vote outcome                                   │  │
│   │ • Signal openness to shareholder input                          │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   REASONS TO FIGHT:                                                     │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Activist demands would destroy value                          │  │
│   │ • Activist candidates are not qualified                         │  │
│   │ • Board has strong support from shareholders                    │  │
│   │ • Company performance strong/improving                          │  │
│   │ • Activist acting in bad faith                                  │  │
│   │ • Precedent concern (invite future activism)                    │  │
│   │ • Activist stake is small                                       │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   TYPICAL SETTLEMENT TERMS:                                             │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • 1-2 board seats (activist or mutually agreed candidates)      │  │
│   │ • Standstill agreement (no further hostile action for period)   │  │
│   │ • Voting agreement (support company nominees)                   │  │
│   │ • Limit on ownership stake                                      │  │
│   │ • Non-disparagement clause                                      │  │
│   │ • Confidentiality provisions                                    │  │
│   │ • Withdrawal of nominations/proposals                           │  │
│   │ • Sometimes: Commitment to strategic review, capital return     │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## Working with Activist Directors

### Best Practices for Activist-Appointed Directors

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    INTEGRATING ACTIVIST DIRECTORS                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   FOR THE BOARD:                                                        │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Welcome as full directors (same rights/responsibilities)      │  │
│   │ • Include on relevant committees based on expertise             │  │
│   │ • Provide comprehensive onboarding                              │  │
│   │ • Foster open dialogue and constructive debate                  │  │
│   │ • Hold accountable to confidentiality and fiduciary duties      │  │
│   │ • Avoid "us vs. them" mentality                                 │  │
│   │ • Listen to perspectives with open mind                         │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   FOR ACTIVIST DIRECTORS:                                               │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ • Understand fiduciary duty is to ALL shareholders              │  │
│   │ • Respect board processes and confidentiality                   │  │
│   │ • Build relationships with fellow directors                     │  │
│   │ • Bring expertise constructively                                │  │
│   │ • Work within the board, not around it                          │  │
│   │ • Avoid back-channel communication to activist                  │  │
│   │ • Focus on long-term value, not short-term agendas              │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   POTENTIAL ISSUES:                                                     │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │ ⚠ Information leakage to activist fund                         │  │
│   │ ⚠ Conflicts of interest on transactions                        │  │
│   │ ⚠ Pressure to prioritize activist agenda                       │  │
│   │ ⚠ Disruption of board dynamics                                 │  │
│   │ ⚠ Short-term focus vs. long-term strategy                      │  │
│   │                                                                 │  │
│   │ MITIGATION:                                                     │  │
│   │ • Clear policies on confidentiality and recusal                 │  │
│   │ • Standstill agreement terms                                    │  │
│   │ • Strong lead independent director                              │  │
│   │ • Regular executive sessions                                    │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## Practice Exercises

### Exercise 1: Vulnerability Assessment

**Scenario:** Analyze this company profile and assess activist vulnerability:

| Factor | Company Data |
|--------|--------------|
| Market cap | $8 billion |
| 3-year TSR | -15% vs. +25% for peers |
| Operating margin | 8% vs. 12% industry average |
| Net cash | $1.5 billion |
| Board tenure | Average 12 years |
| CEO tenure | 8 years |
| Institutional ownership | 85% |

**Questions:**
1. What is the overall vulnerability level?
2. What activist thesis would be most compelling?
3. What proactive steps would you recommend?

### Exercise 2: Response Strategy

**Scenario:** An activist with 7% stake sends a private letter demanding:
- Two board seats
- $2B special dividend
- Sale of underperforming division
- New CEO search

**Develop:**
1. Immediate response plan (first 48 hours)
2. Assessment framework for evaluating demands
3. Engagement strategy
4. Potential settlement terms if applicable

### Exercise 3: Proxy Contest Preparation

**Scenario:** Settlement talks failed. The activist is proceeding with proxy contest to replace three directors (one-third of board).

**Create:**
1. Proxy campaign messaging
2. Shareholder outreach plan
3. Response to activist's public letter
4. ISS/Glass Lewis engagement approach
