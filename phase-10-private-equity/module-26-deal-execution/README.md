# Module 26: Deal Execution

> Mastering the PE deal process from sourcing to close

---

## Overview

This module covers the end-to-end PE deal process. You'll learn how deals are sourced, how to conduct deep due diligence, build financial models, and navigate negotiations through closing.

**Lessons:** 6
**Time:** 5-6 hours
**Difficulty:** Advanced
**Prerequisite:** Module 25 (PE Fundamentals)

---

## Learning Objectives

By completing this module, you will be able to:

1. Source and originate proprietary deal flow
2. Conduct Quality of Earnings analysis
3. Execute commercial and operational due diligence
4. Build comprehensive PE financial models
5. Navigate deal negotiation and closing mechanics

## Key Concepts Preview

### The Deal Funnel

```
┌─────────────────────────────────────────────────────────────┐
│                     PE DEAL FUNNEL                           │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  SOURCING          1000 opportunities reviewed              │
│      │                                                       │
│      ▼                                                       │
│  SCREENING          100 initial evaluations                 │
│      │                                                       │
│      ▼                                                       │
│  PRELIMINARY DD      20 deep dives                          │
│      │                                                       │
│      ▼                                                       │
│  FINAL DD            5 full diligence                       │
│      │                                                       │
│      ▼                                                       │
│  CLOSED              2-3 investments per year               │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### Quality of Earnings Framework

```
REPORTED EBITDA → ADJUSTED EBITDA → PRO FORMA EBITDA

Adjustments include:
• Non-recurring items
• Owner add-backs
• Run-rate adjustments
• Timing differences
• Accounting normalization
```
