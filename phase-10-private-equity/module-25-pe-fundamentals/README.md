# Module 25: Private Equity Fundamentals

> Building the foundation for PE mastery

---

## Overview

This module provides comprehensive grounding in Private Equity fundamentals. You'll learn how the PE industry works, how funds make money, the various investment strategies, and how value is created in leveraged transactions.

**Lessons:** 8
**Time:** 6-8 hours
**Difficulty:** Intermediate

---

## Learning Objectives

By completing this module, you will be able to:

1. Explain the structure of the PE industry and key participants
2. Calculate GP economics including management fees and carried interest
3. Compare different PE investment strategies
4. Apply the three-lever value creation framework
5. Understand LBO mechanics and debt structures
6. Describe the due diligence process
7. Identify key deal structuring considerations
8. Navigate PE career paths and progression

## Key Concepts Preview

### The PE Value Chain
```
FUNDRAISING → DEAL SOURCING → DUE DILIGENCE → EXECUTION
     │              │               │              │
     ▼              ▼               ▼              ▼
  Raise LP      Find target    Validate      Close deal
   capital      companies       thesis

OWNERSHIP → VALUE CREATION → EXIT → RETURNS
     │              │           │        │
     ▼              ▼           ▼        ▼
   Board       Operational   Sale or   Distribute
  oversight    improvement    IPO      to LPs
```

### The Three Value Levers
```
┌─────────────────────────────────────────────────────┐
│                  VALUE CREATION                      │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐ │
│  │ OPERATIONAL │  │  FINANCIAL  │  │  STRATEGIC  │ │
│  │             │  │             │  │             │ │
│  │ • Revenue ↑ │  │ • Leverage  │  │ • M&A       │ │
│  │ • Costs ↓   │  │ • Multiple  │  │ • Market    │ │
│  │ • Working   │  │   expansion │  │   position  │ │
│  │   capital   │  │ • Refinance │  │ • Platform  │ │
│  └─────────────┘  └─────────────┘  └─────────────┘ │
│                                                     │
└─────────────────────────────────────────────────────┘
```
