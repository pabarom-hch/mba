# Module 28: Exit & Returns

> Maximizing value realization and measuring investment performance

---

## Overview

This module covers the critical final phase of the PE investment cycle: exiting portfolio companies and measuring returns. You'll learn how to prepare companies for sale, manage competitive exit processes, analyze value creation attribution, and communicate performance to limited partners.

**Lessons:** 4
**Time:** 4-5 hours
**Difficulty:** Advanced
**Prerequisite:** Module 27 (Operational Improvement)

---

## Learning Objectives

By completing this module, you will be able to:

1. Assess exit readiness and optimize timing decisions
2. Execute a competitive sale process with investment bankers
3. Calculate and attribute investment returns (MOIC, IRR)
4. Prepare comprehensive LP reports and fund performance analysis
