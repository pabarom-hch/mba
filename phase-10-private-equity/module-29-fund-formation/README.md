# Module 29: Fund Formation & Management

> From investment thesis to institutional platform — mastering the GP business

---

## Overview

This module covers the complete fund lifecycle from initial strategy development through launch, governance, and LP relationship management. While Modules 25-28 focus on executing deals and creating value in portfolio companies, Module 29 addresses the **business of running a fund** — how GPs form entities, raise capital, manage governance, and build institutional platforms.

**Lessons:** 6
**Time:** 5-6 hours
**Difficulty:** Advanced
**Prerequisites:** Module 25 (PE Fundamentals), Module 28 (Exit & Returns)

---

## Why Fund Formation & Management?

Understanding the GP business is essential for anyone in the PE ecosystem:

| Dimension | What You'll Learn |
|-----------|-------------------|
| **Fund Strategy** | Develop an investment thesis and select the right vehicle structure |
| **Legal & Regulatory** | Navigate fund documentation, SEC requirements, and compliance |
| **Capital Raising** | Build LP pipelines, manage fundraising processes, and negotiate terms |
| **Fund Operations** | Launch a fund, manage governance, and build institutional infrastructure |
| **LP Relations** | Communicate performance, manage fund lifecycle, and raise successor funds |

### The Fund Launch Formula

The Fund Launch Formula illustrates the iterative cycle of successfully launching a PE fund. This cycle continues until you have sufficient verbal commitments to proceed to legal formation and first close:

```
┌──────────────────────────────────────────────────────────────────┐
│                   THE FUND LAUNCH FORMULA                         │
│              (Iterative Cycle to First Close)                     │

│                    │   thesis        │                           │
│                    │ • Deal pipeline │                           │

│                    └────────┬────────┘                           │
│                             │                                     │
│           ┌─────────────────┼─────────────────┐                  │
│           │                 ▼                 │                  │

│           │    │                 │            │                  │
│           │    │ • Fund design   │            │                  │
│           │    │ • Terms & fees  │            │                  │
│           │    │ • Legal setup   │            │                  │
│           │    └────────┬────────┘            │                  │
│           │             │                     │                  │
│   ITERATE │             ▼                     │ ITERATE          │
│   BASED   │    ┌─────────────────┐            │ BASED            │
│   ON LP   │    │  3. VALIDATION  │            │ ON LP            │
│   FEEDBACK│    │                 │            │ FEEDBACK         │
│           │    │ • Market sound- │            │                  │
│           │    │   ing with LPs  │            │                  │
│           │    │ • Refine thesis │            │                  │
│           │    │ • Test terms    │            │                  │
│           │    └────────┬────────┘            │                  │
│           │             │                     │                  │
│           │             ▼                     │                  │
│           │    ┌─────────────────┐            │                  │
│           │    │   4. CAPITAL    │            │                  │
│           │    │    (VERBAL      │            │                  │

│           ┌─────────────────────────────┐                        │
│           │  THRESHOLD MET?              │                        │
│           │  (Soft-circled ≥ min close) │                        │
│           └─────────────┬───────────────┘                        │
│                         │                                         │
```
