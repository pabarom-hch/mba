# Module 12: Competitive Strategy

## Overview

Competitive strategy determines how a firm positions itself within an industry to achieve sustainable competitive advantage. Building on Michael Porter's foundational work and contemporary strategic thinking, this module develops the analytical skills to understand industry dynamics, competitive positioning, and the sources of superior economic returns.

**Duration:** 2 weeks
**Prerequisites:** Module 1 (Microeconomics), Module 0 (Consulting Mindset)

---

## Learning Objectives

By the end of this module, you will be able to:

1. Apply Porter's Five Forces to analyze industry profitability
2. Identify and evaluate generic competitive strategies
3. Analyze value chain activities for competitive advantage
4. Assess competitive dynamics and strategic interactions
5. Evaluate first-mover advantages and timing of entry
6. Apply game theory to competitive strategy decisions
7. Develop sustainable competitive positions

## Core Concepts

### Sustainable Competitive Advantage

```
REQUIREMENTS FOR SUSTAINABILITY:
─────────────────────────────────────────────────────
1. VALUABLE
   Creates value for customers
   Enables premium pricing or lower costs

2. RARE
   Not possessed by current or potential competitors
   Scarce relative to demand

3. INIMITABLE
   Cannot be easily copied
   - Path dependent
   - Causally ambiguous
   - Socially complex

4. NON-SUBSTITUTABLE
   No strategically equivalent alternatives
   Cannot be circumvented

VRIN Framework = Sustainable Competitive Advantage
```

## Industry Profitability Analysis

### Five Forces Impact on Returns

| Force | Low Threat Conditions | High Threat Conditions |
|-------|----------------------|------------------------|
| Entry | High barriers, retaliation expected | Low barriers, passive incumbents |
| Rivalry | Growth, differentiated, few players | Slow growth, commodity, many players |
| Substitutes | None close, high switching costs | Many alternatives, low switching |
| Buyer Power | Fragmented buyers, high switching | Concentrated, informed, low switching |
| Supplier Power | Many suppliers, standard inputs | Few suppliers, unique inputs |

## Strategic Positioning Examples

### Cost Leadership: Walmart

```
KEY ACTIVITIES:
─────────────────────────────────────────────────────
• Massive scale → purchasing power
• Efficient logistics (cross-docking)
• Low-cost locations
• Minimal store frills
• Technology investment in supply chain
• Everyday Low Pricing (EDLP)

ECONOMIC LOGIC:
─────────────────────────────────────────────────────
Revenue per store: $95M average
Operating margin: ~4%
But: $600B+ in sales = $24B+ operating profit
Competitors cannot match scale advantages
```

### Differentiation: Apple

```
KEY ACTIVITIES:
─────────────────────────────────────────────────────
• Design excellence
• Vertical integration (chip design)
• Ecosystem lock-in
• Premium retail experience
• Brand management
• Controlled distribution

ECONOMIC LOGIC:
─────────────────────────────────────────────────────
iPhone ASP: ~$800 (vs. $250 Android average)
Gross margin: ~45%
Ecosystem switching costs: Very high
Brand premium: 30-50% over comparable specs
```

### Focus: In-N-Out Burger

```
KEY ACTIVITIES:
─────────────────────────────────────────────────────
• Limited menu (4 items)
• Fresh ingredients only
• Regional focus (slow expansion)
• Employee investment
• No franchising
• Cult brand development

ECONOMIC LOGIC:
─────────────────────────────────────────────────────
Per-store sales: 2x McDonald's
Unit economics: Superior
Growth: Intentionally slow
Customer loyalty: Exceptional
```

## Key Formulas and Tools

### Economic Value Created

```
Economic Value = Customer Willingness to Pay - Cost to Produce

Value Captured (Firm) = Price - Cost
Value Captured (Customer) = WTP - Price

Competitive advantage = Capture more value through:
• Higher WTP (differentiation)
• Lower costs (cost leadership)
• Or both (challenging but possible)
```

### Strategic Group Mapping

```
Step 1: Identify key dimensions (2)
Step 2: Plot competitors on map
Step 3: Identify clusters (strategic groups)
Step 4: Analyze mobility barriers between groups
Step 5: Assess attractiveness of each position
```
