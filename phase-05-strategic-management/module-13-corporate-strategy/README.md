# Module 13: Corporate Strategy

## Overview

Corporate strategy addresses the fundamental question: "What businesses should we be in?" While competitive strategy focuses on how to compete within an industry, corporate strategy determines the scope and portfolio of businesses a corporation owns and manages. This module explores diversification, vertical integration, mergers and acquisitions, and how multi-business firms create value beyond what individual businesses could achieve alone.

## Learning Objectives

By the end of this module, you will be able to:

1. Distinguish corporate strategy from business strategy
2. Evaluate diversification decisions using rigorous frameworks
3. Analyze vertical integration trade-offs
4. Assess merger and acquisition logic and execution
5. Apply portfolio management tools to multi-business firms
6. Design corporate parenting strategies that add value
7. Evaluate corporate restructuring and divestiture decisions

## The Diversification Logic

```
WHY DIVERSIFY?
─────────────────────────────────────────────────────
VALID REASONS:              QUESTIONABLE REASONS:
─────────────────────────────────────────────────────
• Economies of scope        • "Diversify risk"
• Transferable capabilities   (shareholders can do)
• Market power             • Empire building
• Internal capital markets  • CEO ego
• Cross-selling            • "Synergy" (undefined)

THE BETTER-OFF TEST:
─────────────────────────────────────────────────────
Does combining these businesses create MORE value
than they would create as separate entities?

If YES → Diversification justified
If NO  → Divest or don't acquire

THE OWNERSHIP TEST:
─────────────────────────────────────────────────────
Are we the BEST owner of this business?

• Do we add unique value?
• Could another parent add more value?
• Should this be independent?
```

## Related vs. Unrelated Diversification

```
RELATED DIVERSIFICATION:
─────────────────────────────────────────────────────
Businesses share resources, capabilities, or markets

Examples:
• Disney: Theme parks + Movies + Streaming + Merchandise
• Apple: Hardware + Software + Services
• Amazon: E-commerce + Cloud + Devices + Logistics

Value creation through:
• Economies of scope
• Knowledge transfer
• Brand leverage
• Cross-selling

UNRELATED DIVERSIFICATION (CONGLOMERATES):
─────────────────────────────────────────────────────
Businesses with no operational connection

Examples:
• Berkshire Hathaway: Insurance + Railroads + Utilities
• Danaher: Life Sciences + Diagnostics + Environmental
• 3M: Industrial + Safety + Consumer + Healthcare

Value creation through:
• Capital allocation
• Turnaround expertise
• Shared services
• Governance excellence

CONGLOMERATE DISCOUNT:
─────────────────────────────────────────────────────
Markets typically value conglomerates at LESS than
sum of parts (10-20% discount typical)

Why?
• Complexity premium demanded
• Agency costs
• Cross-subsidization
• Lack of focus
```

## Key Frameworks

### The BCG Growth-Share Matrix

```
                    RELATIVE MARKET SHARE
                    High              Low
              ┌─────────────────┬─────────────────┐
         High │     STARS       │   QUESTION      │
   MARKET     │                 │     MARKS       │
   GROWTH     │  Invest to      │   Invest        │
   RATE       │  maintain       │   selectively   │
              ├─────────────────┼─────────────────┤
         Low  │   CASH COWS     │      DOGS       │
              │                 │                 │
              │  Harvest/milk   │   Divest or     │
              │  for cash       │   liquidate     │
              └─────────────────┴─────────────────┘

APPLICATION:
• Stars: High growth, high share → Invest to dominate
• Question Marks: High growth, low share → Invest or exit
• Cash Cows: Low growth, high share → Harvest cash
• Dogs: Low growth, low share → Fix, sell, or close
```

## Transaction Costs Framework

```
MAKE VS. BUY DECISION:
─────────────────────────────────────────────────────
                        MARKETS          HIERARCHY
                        (Buy)            (Make)
─────────────────────────────────────────────────────
When?                   • Commodity        • Specific assets
                        • Many suppliers   • Few suppliers
                        • Low uncertainty  • High uncertainty
                        • Simple contract  • Complex needs

Advantages              • Competition      • Control
                        • Flexibility      • Coordination
                        • Efficiency       • Appropriability
                        • Scale           • Secrecy

Disadvantages           • Opportunism      • Bureaucracy
                        • Coordination     • Inflexibility
                        • Hold-up risk     • Higher costs
```

## M&A Success Factors

```
WHY MOST M&A FAILS:
─────────────────────────────────────────────────────
• 70-90% of acquisitions destroy value
• Acquirer's stock typically drops on announcement
• Synergies overestimated, costs underestimated
• Integration challenges underappreciated

CRITICAL SUCCESS FACTORS:
─────────────────────────────────────────────────────
1. STRATEGIC LOGIC
   • Clear rationale beyond "growth"
   • Specific synergies identified
   • Better-off test passed

2. PRICE DISCIPLINE
   • Avoid winner's curse
   • Walk away capability
   • Integration costs included

3. DUE DILIGENCE
   • Thorough investigation
   • Culture assessment
   • Hidden liabilities

4. INTEGRATION PLANNING
   • 100-day plan ready
   • Leadership decisions made
   • Communication strategy

5. EXECUTION
   • Speed and decisiveness
   • Retain key talent
   • Capture synergies quickly
```
