# Strategy Formulation Frameworks

## Learning Objectives

- Apply SWOT analysis to develop strategic options
- Use the Ansoff matrix for growth strategy decisions
- Implement Blue Ocean strategy principles
- Create strategy canvases and value curves
- Develop positioning strategies systematically

---

## Introduction

After analyzing the external environment and internal capabilities, strategists must formulate strategy—choosing where to compete and how to win. This chapter covers the key frameworks for generating strategic options and making positioning choices.

---

## SWOT Analysis

### Framework Structure

```
SWOT MATRIX:
─────────────────────────────────────────────────────

                INTERNAL FACTORS      EXTERNAL FACTORS
         ┌──────────────────────┬──────────────────────┐
         │     STRENGTHS        │   OPPORTUNITIES      │
HELPFUL  │                      │                      │
         │ • What we do well    │ • Favorable trends   │
         │ • Unique resources   │ • Unmet needs        │
         │ • Competitive        │ • Competitor         │
         │   advantages         │   weaknesses         │
         ├──────────────────────┼──────────────────────┤
         │     WEAKNESSES       │     THREATS          │
HARMFUL  │                      │                      │
         │ • What we lack       │ • Unfavorable trends │
         │ • Competitor         │ • New competition    │
         │   advantages         │ • Changing           │
         │ • Internal gaps      │   regulations        │
         └──────────────────────┴──────────────────────┘

INPUTS:
─────────────────────────────────────────────────────
Strengths/Weaknesses ← Internal analysis (VRIO, 7S)
Opportunities/Threats ← External analysis (PESTEL, 5 Forces)
```

### TOWS Strategic Options

```
TOWS MATRIX (Generating Strategies):
─────────────────────────────────────────────────────

               STRENGTHS              WEAKNESSES
         ┌──────────────────────┬──────────────────────┐
         │      S-O             │       W-O            │
OPPORTU- │ Use strengths to     │ Overcome weaknesses  │
NITIES   │ exploit opportunities│ to pursue            │
         │                      │ opportunities        │
         │ MAXI-MAXI            │ MINI-MAXI            │
         │ (Most aggressive)    │ (Turnaround)         │
         ├──────────────────────┼──────────────────────┤
         │      S-T             │       W-T            │
THREATS  │ Use strengths to     │ Minimize weaknesses  │
         │ avoid threats        │ and threats          │
         │                      │                      │
         │ MAXI-MINI            │ MINI-MINI            │
         │ (Defensive)          │ (Survival)           │
         └──────────────────────┴──────────────────────┘

STRATEGY TYPES:
─────────────────────────────────────────────────────
S-O: Aggressive strategy (leverage strengths)
W-O: Developmental strategy (fix weaknesses for opportunity)
S-T: Containment strategy (use strengths to defend)
W-T: Defensive strategy (minimize damage)
```

### SWOT Best Practices

```
COMMON SWOT MISTAKES:
─────────────────────────────────────────────────────
✗ Lists are too long (not prioritized)
✗ Items are vague ("good brand")
✗ Internal/external confused
✗ No connection to strategy
✗ Static analysis (ignores trends)

BEST PRACTICES:
─────────────────────────────────────────────────────
✓ 3-5 items per quadrant (prioritize)
✓ Specific and measurable
✓ Based on evidence (not opinion)
✓ Relative to competitors
✓ Leads to strategic options
✓ Updated regularly
```

## Ansoff Growth Matrix

### Growth Strategy Options

```
ANSOFF MATRIX:
─────────────────────────────────────────────────────
                    PRODUCTS
                Existing        New
           ┌──────────────┬──────────────┐
  Existing │   MARKET     │   PRODUCT    │
           │ PENETRATION  │ DEVELOPMENT  │
           │              │              │
MARKETS    │ Grow share   │ New products │
           │ in current   │ for current  │
           │ market       │ customers    │
           ├──────────────┼──────────────┤
    New    │   MARKET     │ DIVERSI-     │
           │ DEVELOPMENT  │ FICATION     │
           │              │              │
           │ Current      │ New products │
           │ products to  │ for new      │
           │ new markets  │ markets      │
           └──────────────┴──────────────┘

RISK LEVELS:
─────────────────────────────────────────────────────
Market Penetration:   Low risk
Market Development:   Medium risk
Product Development:  Medium risk
Diversification:      High risk
```

### Strategy Details

```
MARKET PENETRATION:
─────────────────────────────────────────────────────
Definition: Increase share in existing markets

Tactics:
• Increase purchase frequency
• Win competitor's customers
• Convert non-users
• Increase usage per customer

When appropriate:
• Market still growing
• Competitors vulnerable
• Customer acquisition cost reasonable

MARKET DEVELOPMENT:
─────────────────────────────────────────────────────
Definition: Take existing products to new markets

Tactics:
• Geographic expansion
• New customer segments
• New channels
• New use cases

When appropriate:
• Strong product/brand
• Transferable capabilities
• New market attractive

PRODUCT DEVELOPMENT:
─────────────────────────────────────────────────────
Definition: New products for existing customers

Tactics:
• Product line extension
• Next-generation products
• Complementary products
• New features/versions

When appropriate:
• Strong customer relationships
• R&D capability
• Customer needs changing

DIVERSIFICATION:
─────────────────────────────────────────────────────
Definition: New products for new markets

Types:
• Related diversification (synergies)
• Unrelated diversification (portfolio)

When appropriate:
• Current markets declining
• Excess resources
• Strategic opportunities

Risk mitigation:
• Start related before unrelated
• Use acquisitions to reduce risk
• Build capabilities first
```

## Blue Ocean Strategy

### Core Concepts

```
BLUE OCEAN FRAMEWORK:
─────────────────────────────────────────────────────

RED OCEAN:                  BLUE OCEAN:
─────────────────────────────────────────────────────
Compete in existing         Create uncontested
market space                market space

Beat the competition        Make competition
                           irrelevant

Exploit existing demand     Create and capture
                           new demand

Make value-cost            Break the value-cost
trade-off                  trade-off

Align activities with      Align activities with
differentiation OR         differentiation AND
low cost                   low cost

CENTRAL IDEA:
─────────────────────────────────────────────────────
Don't compete where everyone is competing
Create new market space where you can win
Make competition irrelevant by changing the game
```

### Four Actions Framework

```
FOUR ACTIONS:
─────────────────────────────────────────────────────

ELIMINATE:
Which factors that the industry takes for granted
should be eliminated?
• Remove features customers don't value
• Challenge industry assumptions

REDUCE:
Which factors should be reduced well below
the industry standard?
• Over-delivered features
• Cost drivers that don't create value

RAISE:
Which factors should be raised well above
the industry standard?
• Underserved customer needs
• New sources of value

CREATE:
Which factors should be created that the
industry has never offered?
• New value propositions
• Unmet needs

ELIMINATE-REDUCE-RAISE-CREATE GRID:
─────────────────────────────────────────────────────
┌─────────────┬─────────────┐
│  ELIMINATE  │   REDUCE    │
├─────────────┼─────────────┤
│    RAISE    │   CREATE    │
└─────────────┴─────────────┘
```

### Strategy Canvas

```
STRATEGY CANVAS:
─────────────────────────────────────────────────────

VALUE           │
High            │
                │    ───────   Industry
                │  ─/        \─
                │ /           \  .... Your strategy
                │/  ....  .....\
                │  .    ..      \.....
                │...
Low             │
                └─────────────────────────────────────
                Factor Factor Factor Factor Factor
                  1      2      3      4      5
                       COMPETING FACTORS

INTERPRETATION:
─────────────────────────────────────────────────────
Industry (solid line): What competitors offer
Your strategy (dotted): How you differentiate

Blue Ocean: Create new curve, not just higher/lower
            on same factors

EXAMPLE: Southwest Airlines
─────────────────────────────────────────────────────
Eliminated: Meals, lounges, seat selection, hub model
Reduced: Ticket prices, turnaround time
Raised: Friendly service, frequency, on-time
Created: Point-to-point travel, car alternative
```

### Blue Ocean Examples

```
CIRQUE DU SOLEIL:
─────────────────────────────────────────────────────
Eliminated: Star performers, animals, aisle sales
Reduced: Danger, humor
Raised: Artistic production, venue elegance
Created: Theme, refined environment, multiple productions

NEW MARKET: Theater + Circus hybrid
Premium pricing, global expansion

YELLOW TAIL WINE:
─────────────────────────────────────────────────────
Eliminated: Wine complexity, aging prestige
Reduced: Wine range, vineyard prestige
Raised: Easy drinking, fun
Created: Social acceptability, "try all"

NEW MARKET: Beer/cocktail drinkers

SOUTHWEST AIRLINES:
─────────────────────────────────────────────────────
Eliminated: Meals, assigned seats, hub connections
Reduced: Price, turnaround time
Raised: Frequency, on-time, service friendliness
Created: Point-to-point convenience

NEW MARKET: Car drivers for short distances
```

## Positioning Strategy

### Strategic Positioning

```
POSITIONING FRAMEWORK:
─────────────────────────────────────────────────────

POSITIONING = How you're perceived relative to
              competitors in the minds of customers

POSITIONING STATEMENT:
─────────────────────────────────────────────────────
For [target segment]
Who [needs/wants]
Our [product/service] is a [category]
That provides [key benefit]
Unlike [alternatives]
Because [reason to believe]

EXAMPLE:
─────────────────────────────────────────────────────
For busy professionals
Who need to stay connected while traveling
The iPhone is a smartphone
That provides seamless productivity and communication
Unlike other devices
Because of its integrated ecosystem and user experience
```

## Key Takeaways

1. **SWOT connects analysis to strategy** - Synthesizes internal/external
2. **TOWS generates strategic options** - Match strengths/weaknesses to opportunities/threats
3. **Ansoff shows growth paths** - And their risk levels
4. **Blue Ocean escapes competition** - Create, don't compete
5. **Four Actions restructure value** - Eliminate, reduce, raise, create
6. **Strategy canvas visualizes differentiation** - Your curve vs. industry
7. **Positioning is perceptual** - About customer's mind
8. **White space is opportunity** - Find unoccupied positions
