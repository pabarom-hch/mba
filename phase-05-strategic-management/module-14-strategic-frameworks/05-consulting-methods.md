# Consulting Methodologies

## Learning Objectives

- Apply McKinsey's problem-solving approach
- Use BCG's strategy frameworks and thinking
- Implement Bain's results-focused methodology
- Master consulting communication and presentation
- Develop the consulting mindset for strategic analysis

---

## Introduction

The top strategy consulting firms—McKinsey, BCG, and Bain (MBB)—have developed distinctive approaches to strategic problem-solving. While the frameworks overlap, each firm has developed signature methodologies that shape how they analyze problems and develop recommendations. Understanding these approaches provides a toolkit for rigorous strategic thinking.

## McKinsey Approach

### Problem-Solving Philosophy

```
McKINSEY PROBLEM-SOLVING:
─────────────────────────────────────────────────────

CORE PRINCIPLES:
─────────────────────────────────────────────────────
1. FACT-BASED: Data drives conclusions
2. HYPOTHESIS-DRIVEN: Start with answer, test
3. ISSUE-FOCUSED: Address what matters
4. STRUCTURED: MECE decomposition
5. ACTION-ORIENTED: Recommendations, not reports

THE McKINSEY MANTRA:
─────────────────────────────────────────────────────
"Facts → Analysis → Recommendations"

Not: "Data → Analysis → Findings"
The difference is action orientation
```

### McKinsey Communication

```
PYRAMID PRINCIPLE (Barbara Minto):
─────────────────────────────────────────────────────
Communication methodology developed at McKinsey

STRUCTURE:
─────────────────────────────────────────────────────
           ┌────────────────────┐
           │  MAIN ANSWER       │
           │  (Start here)      │
           └─────────┬──────────┘
                     │
        ┌────────────┼────────────┐
        │            │            │
   ┌────▼────┐  ┌────▼────┐  ┌────▼────┐
   │ Support │  │ Support │  │ Support │
   │ Point 1 │  │ Point 2 │  │ Point 3 │
   └────┬────┘  └────┬────┘  └────┬────┘
        │            │            │
    Evidence      Evidence     Evidence

RULES:
─────────────────────────────────────────────────────
1. Answer first (don't build to conclusion)
2. Group ideas logically (MECE)
3. Summarize groups (abstraction)
4. Limit to 3-5 points per level
5. Use "so what?" test at each level
```

─────────────────────────────────────────────────────

## Bain Approach

```
BAIN PHILOSOPHY:
─────────────────────────────────────────────────────

CORE PRINCIPLES:
─────────────────────────────────────────────────────
1. RESULTS NOT REPORTS: Measure by client success
2. TRUE NORTH: What's right for client
3. PRAGMATIC: Implementation matters
4. PARTNERSHIP: Long-term relationships
5. TEAM: Collaborative approach

DISTINCTIVE FEATURES:
─────────────────────────────────────────────────────
• Heavy emphasis on implementation
• Private equity expertise
• Net Promoter Score (NPS) inventor
• "True North" values-driven
• Measurable outcomes focus
```

### Bain Frameworks

```
NET PROMOTER SCORE (NPS):
─────────────────────────────────────────────────────
Customer loyalty measurement

Question: "How likely are you to recommend [company]
          to a friend or colleague?" (0-10 scale)

Promoters (9-10):    Loyal enthusiasts
Passives (7-8):      Satisfied but unenthusiastic
Detractors (0-6):    Unhappy customers

NPS = % Promoters - % Detractors

Range: -100 to +100
Good: >0
Excellent: >50
World class: >70

LOYALTY EFFECT:
─────────────────────────────────────────────────────
Higher NPS → Higher growth
5% retention increase → 25-95% profit increase
Promoters are 5x more valuable than detractors

STRATEGIC IMPLICATIONS:
─────────────────────────────────────────────────────
• Focus on creating promoters
• Eliminate detractor drivers
• Customer experience investment justified
```

### Implementation Excellence

```
BAIN IMPLEMENTATION APPROACH:
─────────────────────────────────────────────────────

ELEMENTS FOR RESULTS:
─────────────────────────────────────────────────────
1. Clear accountability
2. Measurable targets
3. Tracking systems
4. Capability building
5. Change management

EXECUTION PREMIUM:
─────────────────────────────────────────────────────
Strategy is only 10% of the challenge
Execution is 90%

Strategy formulation │ Strategy execution
     10%            │       90%
                    │
Research shows most strategies fail in execution
Not because strategy was wrong

BAIN'S RESULTS DELIVERY:
─────────────────────────────────────────────────────
• Define measurable outcomes upfront
• Co-create with client team
• Build client capability
• Stay for implementation
• Tie compensation to results
```

## Consulting Communication

### Presentation Excellence

```
CONSULTING SLIDE PRINCIPLES:
─────────────────────────────────────────────────────

1. ONE MESSAGE PER SLIDE
   • Clear action title (not descriptive)
   • Content supports single point
   • "So what?" obvious

2. ACTION TITLES
   ✗ "Revenue by segment"
   ✓ "Enterprise segment drives 80% of growth"

3. VISUAL HIERARCHY
   • Most important element dominant
   • Eye flows naturally
   • White space used effectively

4. DATA VISUALIZATION
   • Right chart for message
   • Clean, uncluttered
   • Highlighted insights

5. STORYLINE
   • Slides build argument
   • Logical flow
   • Pyramid principle applied
```

### Executive Communication

```
EXECUTIVE SUMMARY STRUCTURE:
─────────────────────────────────────────────────────

1. SITUATION: Context (1-2 sentences)
2. COMPLICATION: Why action needed
3. QUESTION: What we addressed
4. ANSWER: Our recommendation
5. SUPPORT: Key reasons (3 max)
6. IMPLICATIONS: Next steps

EXAMPLE:
─────────────────────────────────────────────────────
SITUATION: Company X has grown 15% annually but
           margins have declined for three years.

COMPLICATION: Without action, profitability will
              fall below cost of capital by 2025.

QUESTION: What should Company X do to restore
          profitable growth?

ANSWER: Exit consumer segment and focus resources
        on enterprise, which has higher margins.

SUPPORT:
• Enterprise margins 25% vs. consumer 5%
• Enterprise growing faster (20% vs. 8%)
• Consumer requires $100M investment to stay competitive

IMPLICATIONS: Divest consumer business (proceeds $200M)
             Invest in enterprise sales and R&D
```

## The Consulting Mindset

### Professional Standards

```
CONSULTING EXCELLENCE:
─────────────────────────────────────────────────────

INTELLECTUAL:
• Rigorous analysis
• Creative problem-solving
• Pattern recognition
• Continuous learning

PROFESSIONAL:
• Client interest first
• Confidentiality
• Objectivity
• Intellectual honesty

INTERPERSONAL:
• Active listening
• Clear communication
• Team collaboration
• Influence without authority

EXECUTION:
• Attention to detail
• Time management
• Follow-through
• Quality standards
```

### Consulting Survival Skills

```
KEY HABITS:
─────────────────────────────────────────────────────

1. ASK "SO WHAT?" constantly
   Every analysis must have implications

2. ISSUE, NOT TASK oriented
   Focus on what matters, not what's asked

3. 80/20 ruthlessly
   Perfect is enemy of good

4. ELEVATOR TEST
   Explain anything in 30 seconds

5. DISAGREE AND COMMIT
   Voice concerns, then execute

6. BE HELPFUL
   Success is team success

7. LEARN QUICKLY
   Become expert fast

8. STAY SKEPTICAL
   Question everything (including yourself)
```

## Key Takeaways

1. **McKinsey: Structured problem-solving** - MECE, hypothesis-driven
2. **BCG: Economic rigor** - Experience curve, portfolio, time-based
3. **Bain: Results delivery** - Implementation, NPS, measurable outcomes
4. **Pyramid Principle: Answer first** - Structure communication clearly
5. **Action titles: Messages, not labels** - Every slide makes a point
6. **Execution matters: 90% of battle** - Strategy without execution fails
7. **Consulting mindset: Continuous improvement** - Learning and adaptation
8. **Client focus: True North** - What's right for client

## Practice Problems

### Problem 1: Pyramid Structure
Structure a recommendation to enter a new market using the pyramid principle. Create the top-level message and three supporting arguments.

### Problem 2: Experience Curve
A company has 25% market share. Calculate how much its costs might decline if it doubled share, assuming 20% learning rate.

### Problem 3: NPS Strategy
A company has NPS of +15 with 40% promoters, 35% passives, 25% detractors. Develop a strategy to reach +40 NPS.
