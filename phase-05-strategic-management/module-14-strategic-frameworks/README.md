# Module 14: Strategic Analysis Frameworks

## Overview

Strategic analysis frameworks are the intellectual tools consultants and strategists use to structure complex business problems. From McKinsey's 7S model to BCG's growth-share matrix, these frameworks provide disciplined approaches to diagnosing strategic situations, evaluating options, and making recommendations. This capstone module synthesizes the major frameworks and teaches when and how to apply them effectively.

## Learning Objectives

By the end of this module, you will be able to:

1. Select appropriate frameworks for different strategic situations
2. Apply McKinsey, BCG, and Bain frameworks professionally
3. Structure strategic analyses systematically
4. Combine multiple frameworks for comprehensive analysis
5. Present strategic recommendations with rigor and clarity
6. Critique and adapt frameworks for specific contexts
7. Develop custom frameworks when standard tools don't fit

## The Consulting Approach

```
McKINSEY PROBLEM-SOLVING PROCESS:
─────────────────────────────────────────────────────
1. DEFINE THE PROBLEM
   • What is the key question?
   • What would success look like?
   • What is the scope?

2. STRUCTURE THE ANALYSIS
   • Break into components (MECE)
   • Identify key drivers
   • Develop hypothesis

3. PRIORITIZE
   • Which analyses matter most?
   • 80/20 focus
   • Quick wins vs. deep dives

4. GATHER EVIDENCE
   • Data collection
   • Interviews
   • Benchmarking

5. SYNTHESIZE
   • "So what?" conclusions
   • Pattern recognition
   • Insight generation

6. RECOMMEND
   • Actionable recommendations
   • Implementation plan
   • Risk mitigation

BCG STRATEGY APPROACH:
─────────────────────────────────────────────────────
• "Competition on capabilities"
• Time-based competition
• Experience curve economics
• Portfolio optimization

BAIN RESULTS FOCUS:
─────────────────────────────────────────────────────
• Focus on implementation
• Measurable outcomes
• Client capability building
• Long-term relationships
```

## MECE Principle

```
MUTUALLY EXCLUSIVE, COLLECTIVELY EXHAUSTIVE:
─────────────────────────────────────────────────────

MUTUALLY EXCLUSIVE:
• No overlap between categories
• Each element fits one bucket only
• Clean boundaries

COLLECTIVELY EXHAUSTIVE:
• All possibilities covered
• Nothing left out
• Complete picture

EXAMPLE - REVENUE GROWTH:
─────────────────────────────────────────────────────
MECE breakdown:
┌─────────────────────────────────────────────────┐
│ Revenue Growth                                   │
├─────────────────────────────────────────────────┤
│ ├── Price (increase price per unit)             │
│ │   ├── Existing products                       │
│ │   └── Premium products/services               │
│ ├── Volume (sell more units)                    │
│ │   ├── Existing customers (cross-sell/upsell) │
│ │   ├── New customers (same market)            │
│ │   └── New markets (geographic/segment)       │
│ └── Mix (shift to higher-margin products)      │
│     ├── Product mix                             │
│     └── Customer mix                            │
└─────────────────────────────────────────────────┘
```

## Key Frameworks Reference

### McKinsey 7S Model

```
McKINSEY 7S FRAMEWORK:
─────────────────────────────────────────────────────
            ┌───────────────┐
            │   STRATEGY    │
            └───────┬───────┘
                    │
    ┌───────────────┼───────────────┐
    │               │               │
┌───▼───┐     ┌─────▼─────┐     ┌───▼───┐
│STRUCTURE│◄──►│  SHARED   │◄──►│SYSTEMS│
└───┬───┘     │  VALUES   │     └───┬───┘
    │         └─────┬─────┘         │
    │               │               │
    └───────────────┼───────────────┘
                    │
    ┌───────────────┼───────────────┐
    │               │               │
┌───▼───┐     ┌─────▼─────┐     ┌───▼───┐
│ STYLE │◄──►│   STAFF   │◄──►│SKILLS │
└───────┘     └───────────┘     └───────┘

HARD S's (easier to change):
• Strategy, Structure, Systems

SOFT S's (harder to change):
• Style, Staff, Skills, Shared Values
```

### VRIO Framework

```
VRIO ANALYSIS:
─────────────────────────────────────────────────────
Does resource/capability provide advantage?

V - VALUABLE?
    Does it allow exploitation of opportunity
    or neutralization of threat?
    NO → Competitive disadvantage

R - RARE?
    Is it controlled by few firms?
    NO → Competitive parity

I - INIMITABLE?
    Is it costly to imitate?
    NO → Temporary advantage

O - ORGANIZED?
    Is firm organized to exploit it?
    NO → Unused potential
    YES → SUSTAINABLE COMPETITIVE ADVANTAGE
```

### Blue Ocean Strategy

```
BLUE OCEAN FRAMEWORK:
─────────────────────────────────────────────────────
RED OCEAN:               BLUE OCEAN:
• Compete in existing    • Create new market space
• Beat competition       • Make competition irrelevant
• Exploit existing       • Create and capture new
  demand                   demand
• Value-cost trade-off   • Break value-cost trade-off
• Align activities       • Align activities with
  with differentiation     differentiation AND low cost
  OR low cost

FOUR ACTIONS FRAMEWORK:
─────────────────────────────────────────────────────
ELIMINATE: Which factors can be eliminated?
REDUCE: Which factors can be reduced below standard?
RAISE: Which factors can be raised above standard?
CREATE: Which factors never offered can be created?

STRATEGY CANVAS:
─────────────────────────────────────────────────────
Plot competitor offerings on key factors
Identify differentiation opportunity
Create new value curve
```
