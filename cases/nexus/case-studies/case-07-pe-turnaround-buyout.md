## Part 5: Employee Ownership Structures

### Three ESOP Approaches (Comparative Analysis)

This section compares three employee ownership structures. All three achieve the goal of aligning employees with exit value, but with different tradeoffs.

---

### Option A: Traditional ESOP (Tax-Advantaged Trust)

```
TRADITIONAL ESOP STRUCTURE
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│ STRUCTURE                                                                   │
│ ├── ESOP Trust purchases 15% of company stock                              │
│ ├── Company makes tax-deductible contributions to trust                    │
│ ├── Shares allocated to employees based on compensation/tenure             │
│ └── Employees vest over 3-6 years                                          │
│                                                                             │
│ TAX ADVANTAGES                                                              │
│ ├── Company: Contributions are tax-deductible                              │
│ ├── Employees: No tax until distribution (retirement/departure)            │
│ ├── S-Corp: ESOP-owned portion not subject to federal income tax           │
│ └── Seller: 1042 rollover defers capital gains (C-Corp only)               │
│                                                                             │
│ GOVERNANCE                                                                  │
│ ├── Trustee manages shares on behalf of employees                          │
│ ├── Employees don't vote shares directly (trustee votes)                   │
│ ├── Annual valuation required (409A compliance)                            │
│ └── ERISA fiduciary requirements apply                                     │
│                                                                             │
│ COSTS                                                                       │
│ ├── Setup: $50-100K (legal, valuation, trustee)                            │
│ ├── Annual: $15-30K (valuation, administration, audit)                     │
│ └── Complexity: HIGH (ERISA compliance, DOL oversight)                     │
│                                                                             │
│ BEST FOR                                                                    │
│ └── Companies planning long-term independence (not quick PE exit)          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Option B: Direct Equity Grants (Options/RSUs)

```
DIRECT EQUITY STRUCTURE
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│ STRUCTURE                                                                   │
│ ├── Create option pool (15% of fully-diluted equity)                       │
│ ├── Grant options/RSUs to employees individually                           │
│ ├── 4-year vesting with 1-year cliff (standard)                            │
│ └── Exercise at liquidity event (cashless exercise typical)                │
│                                                                             │
│ GRANT TYPES                                                                 │
│ ├── ISOs (Incentive Stock Options)                                         │
│ │   └── Tax-advantaged for employees, $100K/year limit                     │
│ ├── NSOs (Non-Qualified Stock Options)                                     │
│ │   └── No limit, but ordinary income tax at exercise                      │
│ ├── RSUs (Restricted Stock Units)                                          │
│ │   └── Simpler, value at vesting taxed as ordinary income                 │
│ └── Profits Interests (LLC structure)                                      │
│     └── Capital gains treatment, participation above threshold             │
│                                                                             │
│ GOVERNANCE                                                                  │
│ ├── Board approves grants (compensation committee)                         │
│ ├── Individual grant agreements                                            │
│ ├── 409A valuation required for strike price                               │
│ └── Cap table management (Carta, Pulley, etc.)                             │
│                                                                             │
│ COSTS                                                                       │
│ ├── Setup: $20-40K (legal, 409A valuation)                                 │
│ ├── Annual: $5-15K (valuation refresh, administration)                     │
│ └── Complexity: MEDIUM (individual grants, tracking)                       │
│                                                                             │
│ BEST FOR                                                                    │
│ └── PE-backed companies planning exit (most common PE model)               │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Option C: Employee Stock Purchase Plan (ESPP)

```
ESPP STRUCTURE
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│ STRUCTURE                                                                   │
│ ├── Employees purchase shares at discount (typically 15%)                  │
│ ├── Payroll deductions fund purchases (1-10% of salary)                    │
│ ├── Purchase periods (6 months typical)                                    │
│ └── Lookback provision: discount on lower of start/end price              │
│                                                                             │
│ TAX TREATMENT                                                               │
│ ├── Qualified (423 Plan): Capital gains if held 2+ years                   │
│ ├── Non-Qualified: Discount taxed as ordinary income                       │
│ └── Company: No deduction on qualified plans                               │
│                                                                             │
│ GOVERNANCE                                                                  │
│ ├── Shareholder approval required for 423 plans                            │
│ ├── Must be offered to all employees (non-discriminatory)                  │
│ ├── Annual limits: $25K fair market value per year                         │
│ └── Administration through payroll system                                  │
│                                                                             │
│ COSTS                                                                       │
│ ├── Setup: $15-30K (legal, plan document)                                  │
│ ├── Annual: $10-20K (administration, record-keeping)                       │
│ └── Complexity: LOW-MEDIUM                                                 │
│                                                                             │
│ BEST FOR                                                                    │
│ └── Public companies or companies with regular liquidity                   │
│                                                                             │
│ LIMITATION FOR PE                                                           │
│ └── Limited utility in private company (no market for shares)              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Common Student Mistakes

1. **Over-leveraging**: Using too much debt for a services business with variable cash flows
2. **Aggressive multiples**: Assuming multiple expansion without earning it through performance
3. **Integration complexity**: Underestimating M&A integration challenges
4. **Employee communication**: Treating equity as purely transactional vs. cultural
5. **Exit timing**: Waiting too long or rushing before value creation is complete

## Appendix: Key Terms

| Term | Definition |
|------|------------|
| **LBO** | Leveraged Buyout - acquisition using significant debt |
| **MOIC** | Multiple on Invested Capital - total return multiple |
| **IRR** | Internal Rate of Return - annualized return percentage |
| **Tuck-in** | Small acquisition integrated into platform company |
| **ESOP** | Employee Stock Ownership Plan - tax-advantaged trust |
| **Phantom Equity** | Cash-settled equity substitute, no actual shares |
| **Waterfall** | Distribution order of proceeds at exit |
| **409A Valuation** | IRS-compliant fair market value for equity grants |
| **Seller Note** | Deferred payment to seller, subordinated to senior debt |
| **Management Rollover** | Existing management reinvests proceeds into new deal |
