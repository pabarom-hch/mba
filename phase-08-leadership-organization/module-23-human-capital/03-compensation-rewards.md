## Base Pay Structure

### Job Evaluation Methods

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    JOB EVALUATION APPROACHES                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   1. MARKET PRICING (Most Common)                                           │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Match jobs to market survey data                                │   │
│   │   • Pay based on external market rates                              │   │
│   │   • Best for: Common jobs with clear market comparables             │   │
│   │                                                                     │   │
│   │   Process:                                                          │   │
│   │   Job Match → Survey Data → Market Rate → Pay Decision              │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   2. POINT-FACTOR METHOD                                                    │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Evaluate jobs on compensable factors                            │   │
│   │   • Assign points based on factor levels                            │   │
│   │   • Total points determine job grade                                │   │
│   │   • Best for: Internal equity focus, unique jobs                    │   │
│   │                                                                     │   │
│   │   Common Factors:                                                   │   │
│   │   • Knowledge/Skills (30%)                                          │   │
│   │   • Problem Solving (25%)                                           │   │
│   │   • Accountability (30%)                                            │   │
│   │   • Working Conditions (15%)                                        │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   3. JOB CLASSIFICATION                                                     │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Define grade levels with descriptions                           │   │
│   │   • Slot jobs into appropriate grades                               │   │
│   │   • Best for: Government, simple structures                         │   │
│   │                                                                     │   │
│   │   Example: GS levels in federal government                          │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Key Compensation Metrics

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    COMPENSATION METRICS                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   COMPA-RATIO                                                               │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │                                                                     │   │
│   │   Compa-Ratio = Actual Salary / Midpoint                            │   │
│   │                                                                     │   │
│   │   INTERPRETATION:                                                   │   │
│   │   • <0.80 = Underpaid (new to role, developing)                     │   │
│   │   • 0.80-0.90 = Below midpoint (still learning)                     │   │
│   │   • 0.90-1.10 = At market (fully competent)                         │   │
│   │   • 1.10-1.20 = Above midpoint (high performer)                     │   │
│   │   • >1.20 = Overpaid (red circle - at max)                          │   │
│   │                                                                     │   │
│   │   Example: Salary $85K, Midpoint $90K → CR = 0.94                   │   │
│   │                                                                     │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   RANGE PENETRATION                                                         │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │                                                                     │   │
│   │   Range Penetration = (Salary - Min) / (Max - Min)                  │   │
│   │                                                                     │   │
│   │   Shows where employee sits within the entire range                 │   │
│   │   Example: Salary $85K, Min $70K, Max $110K                         │   │
│   │   RP = ($85K - $70K) / ($110K - $70K) = 37.5%                       │   │
│   │                                                                     │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   MARKET RATIO                                                              │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │                                                                     │   │
│   │   Market Ratio = Actual Salary / Market Rate                        │   │
│   │                                                                     │   │
│   │   Indicates competitiveness vs. external market                     │   │
│   │                                                                     │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Equity Compensation

### Types of Equity

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    EQUITY COMPENSATION TYPES                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   STOCK OPTIONS                                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   Right to purchase shares at a fixed "strike" price                │   │
│   │                                                                     │   │
│   │   Example: 1,000 options, strike price $10                          │   │
│   │   Stock rises to $50 → Value = ($50 - $10) x 1,000 = $40,000        │   │
│   │   Stock falls to $5 → Options "underwater" (worthless)              │   │
│   │                                                                     │   │
│   │   Types:                                                            │   │
│   │   • ISO (Incentive Stock Options): Tax advantaged, employees only   │   │
│   │   • NSO (Non-Qualified): Taxed as income, can give to anyone        │   │
│   │                                                                     │   │
│   │   Vesting: Typically 4 years with 1-year cliff                      │   │
│   │   │  Year 1: 25%  │  Year 2: 50%  │  Year 3: 75%  │  Year 4: 100%  │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   RESTRICTED STOCK UNITS (RSUs)                                             │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   Promise to deliver shares upon vesting                            │   │
│   │                                                                     │   │
│   │   Example: 500 RSUs granted when stock = $50                        │   │
│   │   Stock rises to $100 → Value = 500 x $100 = $50,000                │   │
│   │   Stock falls to $25 → Value = 500 x $25 = $12,500 (still value!)   │   │
│   │                                                                     │   │
│   │   Pros: Simpler than options; always have value                     │   │
│   │   Cons: Less upside leverage; taxed as income at vesting            │   │
│   │                                                                     │   │
│   │   Common in public companies; increasingly in private               │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   PERFORMANCE STOCK UNITS (PSUs)                                            │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   RSUs that vest based on performance conditions                    │   │
│   │                                                                     │   │
│   │   Example conditions:                                               │   │
│   │   • Relative TSR (Total Shareholder Return)                         │   │
│   │   • Revenue growth                                                  │   │
│   │   • EPS targets                                                     │   │
│   │                                                                     │   │
│   │   Payout range: 0% to 200% of target shares                         │   │
│   │   Popular for executives; aligns with shareholder value             │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│   ESPP (Employee Stock Purchase Plan)                                       │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   Employees can purchase stock at a discount (typically 15%)        │   │
│   │                                                                     │   │
│   │   Broad-based participation; popular employee benefit               │   │
│   │   Tax-advantaged under IRC Section 423                              │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### Equity Philosophy by Company Stage

| Stage | Typical Approach | Key Considerations |
|-------|------------------|-------------------|
| **Startup** | Heavy options, low salary | High risk/reward; retention critical |
| **Growth** | Mix of options/RSUs | Balancing cash vs. equity; dilution |
| **Public** | RSUs, PSUs, ESPP | Shareholder alignment; liquidity |
| **Mature** | Cash-heavy, some equity | Stability; predictable costs |

## Practice Exercise: Total Rewards Design

### Scenario

You are the VP of Total Rewards at a 1,000-person B2B SaaS company that recently went public. The company has grown rapidly (from 200 employees 3 years ago) but compensation practices have been inconsistent, leading to:

- High turnover in engineering (30%)
- Pay equity issues discovered in recent audit (women paid 94 cents on dollar)
- Employees confused about their total rewards package
- Sales team unhappy with commission structure

**Current State:**
- Base pay: No formal structure; manager discretion
- Variable pay: 10% discretionary bonus for all; sales has complex commission
- Equity: Stock options granted inconsistently at hire only
- Benefits: Basic package, below market

**Budget:** You have $5M additional budget for compensation improvements

### Design Requirements

1. **Base Pay Structure**
   - Design a job leveling framework
   - Recommend market positioning strategy
   - Address pay equity issues

2. **Variable Pay Redesign**
   - Propose annual bonus plan design
   - Recommend sales compensation structure changes
   - Include metrics and targets

3. **Equity Strategy**
   - Recommend ongoing equity program (refreshers)
   - Propose vesting schedule
   - Address fairness in existing grants

4. **Benefits Enhancement**
   - Prioritize benefit improvements within budget
   - Consider what will most impact retention

5. **Communication Plan**
   - How will you explain changes to employees?
   - How will you train managers?
