## Taking Action on Engagement

### Survey to Action Process

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    ENGAGEMENT ACTION CYCLE                                  │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   1. SURVEY                                                                 │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Design and administer survey                                    │   │
│   │   • Ensure high participation                                       │   │
│   │   • Maintain confidentiality                                        │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│          │                                                                  │
│          ▼                                                                  │
│   2. ANALYZE                                                                │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Review overall results                                          │   │
│   │   • Identify strengths and opportunities                            │   │
│   │   • Compare to benchmarks                                           │   │
│   │   • Analyze by segment (dept, location, tenure)                     │   │
│   │   • Look for correlations                                           │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│          │                                                                  │
│          ▼                                                                  │
│   3. SHARE                                                                  │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Thank employees for participation                               │   │
│   │   • Share results transparently                                     │   │
│   │   • Acknowledge both positives and concerns                         │   │
│   │   • Commit to action                                                │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│          │                                                                  │
│          ▼                                                                  │
│   4. PLAN                                                                   │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Identify priority focus areas (2-3 max)                         │   │
│   │   • Involve employees in solution design                            │   │
│   │   • Create specific action plans                                    │   │
│   │   • Assign owners and timelines                                     │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│          │                                                                  │
│          ▼                                                                  │
│   5. ACT                                                                    │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Implement changes                                               │   │
│   │   • Communicate progress regularly                                  │   │
│   │   • Remove barriers                                                 │   │
│   │   • Celebrate wins                                                  │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│          │                                                                  │
│          ▼                                                                  │
│   6. FOLLOW UP                                                              │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │   • Pulse check on progress                                         │   │
│   │   • Adjust approach as needed                                       │   │
│   │   • Measure impact                                                  │   │
│   │   • Repeat cycle                                                    │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Practice Exercise: Engagement Strategy Design

### Scenario

You are the VP of People at a 2,000-person technology company. The recent engagement survey revealed:

**Survey Results:**
- Overall engagement: 62% favorable (down from 68% last year)
- eNPS: +15 (down from +25)
- Participation: 78%

**Lowest Scoring Areas:**
- Career growth opportunities: 52%
- Recognition: 55%
- Trust in leadership: 58%
- Work-life balance: 60%

**Additional Context:**
- Voluntary turnover: 22% (industry average: 18%)
- High performer turnover: 28%
- Manager scores vary widely (45% to 85% by team)
- Recent acquisition (6 months ago) created uncertainty

### Design a comprehensive engagement improvement strategy:

1. **Analysis**
   - What are the root causes behind these results?
   - What additional data would you want?
   - How would you segment the analysis?

2. **Communication**
   - How will you share results with employees?
   - What will you say about the decline?
   - How will you rebuild trust?

3. **Company-Wide Initiatives**
   - What 2-3 company-wide actions will you take?
   - How will you address the acquisition uncertainty?
   - What quick wins can you achieve?

4. **Manager-Led Action**
   - How will you enable managers to act on their team results?
   - What tools and training will you provide?
   - How will you address manager variability?

5. **Retention Focus**
   - What specific actions for high performers?
   - How will you identify flight risks?
   - What retention investments will you make?

6. **Measurement**
   - How will you track progress?
   - When will you pulse check?
   - What leading indicators will you monitor?
