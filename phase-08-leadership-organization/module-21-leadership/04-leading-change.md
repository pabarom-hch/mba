# Lesson 4: Leading Change

## Module 21: Leadership | Estimated Time: 3 Hours

---

## Learning Objectives

By the end of this lesson, you will be able to:

1. **Apply** Kotter's 8-step change model to organizational transformations
2. **Recognize** stages of the change curve and manage emotional responses
3. **Identify** sources of resistance and develop strategies to address them
4. **Design** a change management plan for a significant initiative
5. **Communicate** effectively throughout the change process

---

## Introduction

Change is the only constant in business, yet most change initiatives fail. Research consistently shows that 70% of organizational change efforts do not achieve their intended outcomes. This failure is rarely due to flawed strategy but rather to poor change leadership. Understanding how to lead change effectively is one of the most critical leadership competencies.

### Managing Each Stage

| Stage | Characteristics | Leadership Actions |
|-------|-----------------|-------------------|
| 1. Shock | Surprise, immobilization | Give time to absorb, provide facts |
| 2. Denial | "This won't really happen" | Continue communicating, be patient |
| 3. Anger | Frustration, blame, resistance | Listen, acknowledge feelings, don't argue |
| 4. Depression | Low energy, uncertainty | Support, coach, show empathy |
| 5. Experimenting | Trying new behaviors | Encourage, celebrate small wins |
| 6. Decision | Commitment to new way | Reinforce, provide resources |
| 7. Integration | New norm established | Recognize achievement, anchor in culture |

## 4. Change Communication Framework

### The ADKAR Model

```
┌─────────────────────────────────────────────────────────────────────┐
│              THE ADKAR MODEL FOR CHANGE                              │
│              (Prosci)                                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  Sequential building blocks for individual change:                   │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  A ─── AWARENESS of the need for change                     │    │
│  │        "Why is change necessary?"                           │    │
│  │                       │                                      │    │
│  │                       ▼                                      │    │
│  │  D ─── DESIRE to participate and support the change         │    │
│  │        "What's in it for me?"                               │    │
│  │                       │                                      │    │
│  │                       ▼                                      │    │
│  │  K ─── KNOWLEDGE of how to change                           │    │
│  │        "What do I need to know?"                            │    │
│  │                       │                                      │    │
│  │                       ▼                                      │    │
│  │  A ─── ABILITY to implement the change                      │    │
│  │        "Can I actually do this?"                            │    │
│  │                       │                                      │    │
│  │                       ▼                                      │    │
│  │  R ─── REINFORCEMENT to sustain the change                  │    │
│  │        "How do we make it stick?"                           │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  DIAGNOSTIC USE: Identify where individuals are stuck to target     │
│  interventions appropriately                                         │
│                                                                       │
│  COMMON STICKING POINTS:                                             │
│  • Awareness → Desire (people understand but don't want to)         │
│  • Knowledge → Ability (trained but can't apply)                    │
│  • Ability → Reinforcement (doing it but reverting)                 │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

## Practice Exercises

### Exercise 1: Change Diagnosis

Think of a change initiative in your organization (past or present):

1. At which Kotter step did it succeed or fail?
2. What resistance factors were present?
3. How were they addressed (or not)?
4. What would you do differently?

### Exercise 2: Urgency Assessment

Rate your organization on these urgency indicators (1-5):

| Indicator | Score |
|-----------|-------|
| People acknowledge need for change | |
| External perspective is valued | |
| Complacency is actively challenged | |
| Bold targets are set | |
| Fear of status quo exceeds fear of change | |

If total < 15, urgency is insufficient for major change.

### Exercise 3: Coalition Building

For a change you want to lead:

1. Who must be in the guiding coalition? (List 5-7 people)
2. What does each bring (position power, expertise, credibility)?
3. Who are key influencers you need to convert?
4. How will you build trust and commitment within the coalition?
