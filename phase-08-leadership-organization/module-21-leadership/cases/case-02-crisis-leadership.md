## Case A: Johnson & Johnson Tylenol Crisis (1982)

### The Crisis

In September 1982, seven people in the Chicago area died after taking Tylenol capsules that had been laced with cyanide by an unknown person. Tylenol was the nation's leading over-the-counter pain reliever, with 35% market share and $400 million in annual sales.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    TYLENOL CRISIS TIMELINE                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  September 29, 1982                                                  │
│  └── First deaths reported in Chicago area                          │
│                                                                       │
│  September 30, 1982                                                  │
│  └── Link to Tylenol capsules established                           │
│                                                                       │
│  October 1-5, 1982                                                   │
│  └── J&J leadership response begins                                 │
│      • CEO James Burke takes personal control                       │
│      • Nationwide recall initiated (31 million bottles)             │
│      • Production halted                                            │
│      • $100 million recall cost absorbed                            │
│                                                                       │
│  October 1982 - February 1983                                        │
│  └── Recovery and innovation                                        │
│      • Triple-seal tamper-resistant packaging developed             │
│      • Tylenol relaunched                                           │
│      • Market share recovered within one year                       │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  KEY FACT: J&J was NOT responsible for the tampering.      │    │
│  │  The crime was committed by an external individual.         │    │
│  │  Yet J&J took full responsibility for consumer safety.      │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Leadership Response

CEO James Burke's leadership during the crisis became the gold standard for crisis management:

```
┌─────────────────────────────────────────────────────────────────────┐
│              JAMES BURKE'S CRISIS LEADERSHIP                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  GUIDING PRINCIPLE: The J&J Credo                                    │
│  ─────────────────────────────────                                   │
│  "Our first responsibility is to the doctors, nurses, patients,     │
│   mothers and fathers and all others who use our products."         │
│                                                                       │
│  Burke's decision: If the Credo meant anything, they had to         │
│  put customer safety above short-term profits.                       │
│                                                                       │
│  KEY LEADERSHIP ACTIONS:                                             │
│                                                                       │
│  1. TOOK PERSONAL OWNERSHIP                                          │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Burke personally led the response                      │   │
│     │  • Established 7-member strategy team                    │   │
│     │  • Available 24/7 for media and stakeholders            │   │
│     │  • Made himself the face of the company                  │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  2. PRIORITIZED SAFETY OVER PROFITS                                  │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Ordered immediate nationwide recall                   │   │
│     │  • Halted all Tylenol production                        │   │
│     │  • Cost: $100+ million                                   │   │
│     │  • Never questioned decision despite financial impact    │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  3. COMMUNICATED TRANSPARENTLY                                       │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Held press conferences daily                          │   │
│     │  • Shared all available information openly               │   │
│     │  • Warned consumers not to use product                   │   │
│     │  • Established toll-free hotline (1-800 number)         │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  4. INNOVATED FOR THE FUTURE                                         │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Developed triple-seal tamper-resistant packaging      │   │
│     │  • Set new industry safety standard                      │   │
│     │  • Turned crisis into competitive advantage             │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  RESULT: Tylenol recovered its market leadership within one year    │
│  and the case became the textbook example of crisis leadership.     │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Why It Worked

| Factor | Implementation |
|--------|---------------|
| Values-Based Decision Making | The Credo provided clear guidance for prioritizing stakeholders |
| Speed | Decisions made within hours, not days |
| Transparency | Open communication, even when news was bad |
| Accountability | Never blamed others or made excuses |
| Humanity | Put human safety above all other concerns |
| Innovation | Used crisis as catalyst for improvement |

## Case B: BP Deepwater Horizon Crisis (2010)

### The Crisis

On April 20, 2010, the Deepwater Horizon drilling rig exploded in the Gulf of Mexico, killing 11 workers and creating the largest marine oil spill in history. The well leaked for 87 days, releasing an estimated 4.9 million barrels of oil.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    DEEPWATER HORIZON TIMELINE                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  April 20, 2010                                                      │
│  └── Explosion on Deepwater Horizon; 11 workers killed              │
│                                                                       │
│  April 22, 2010                                                      │
│  └── Rig sinks; oil begins leaking from wellhead                    │
│                                                                       │
│  April - July 2010                                                   │
│  └── Failed attempts to cap the well                                │
│      • "Top kill" fails                                             │
│      • "Junk shot" fails                                            │
│      • Multiple containment efforts fail                            │
│                                                                       │
│  July 15, 2010                                                       │
│  └── Well finally capped after 87 days                              │
│                                                                       │
│  IMPACT:                                                             │
│  • 4.9 million barrels of oil released                              │
│  • 11 lives lost                                                    │
│  • Environmental catastrophe affecting Gulf ecosystem               │
│  • Economic devastation for Gulf communities                        │
│  • $65+ billion in costs and settlements                            │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Leadership Failures

CEO Tony Hayward's response became a case study in how NOT to lead during a crisis:

```
┌─────────────────────────────────────────────────────────────────────┐
│              TONY HAYWARD'S CRISIS LEADERSHIP FAILURES               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  FAILURE 1: MINIMIZING THE CRISIS                                    │
│  ─────────────────────────────────                                   │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  "The Gulf of Mexico is a very big ocean. The amount of    │    │
│  │   volume of oil and dispersant we are putting into it is   │    │
│  │   tiny in relation to the total water volume."             │    │
│  │                                                              │    │
│  │  → Dismissed environmental concerns while oil continued     │    │
│  │    to devastate coastal ecosystems and communities          │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  FAILURE 2: MAKING IT ABOUT HIMSELF                                  │
│  ──────────────────────────────────                                  │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  "There's no one who wants this over more than I do.       │    │
│  │   I'd like my life back."                                   │    │
│  │                                                              │    │
│  │  → While 11 workers had lost their lives and thousands     │    │
│  │    were suffering, this statement showed stunning lack      │    │
│  │    of empathy                                               │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  FAILURE 3: DEFLECTING BLAME                                         │
│  ───────────────────────────                                         │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  "This was not our accident... this was Transocean's rig.  │    │
│  │   Their systems. Their people. Their equipment."           │    │
│  │                                                              │    │
│  │  → Attempted to shift responsibility to contractors        │    │
│  │    rather than accepting accountability                     │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  FAILURE 4: POOR VISIBILITY                                          │
│  ──────────────────────────                                          │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  • Hayward attended a yacht race while oil continued       │    │
│  │    to leak                                                  │    │
│  │  • Created perception of being disconnected from crisis    │    │
│  │  • Symbolized corporate indifference                       │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  RESULT: Hayward was removed as CEO. BP's reputation suffered       │
│  lasting damage. The response became a textbook example of          │
│  crisis leadership failure.                                          │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### What Went Wrong

| Factor | Failure |
|--------|---------|
| Empathy | Self-focused comments instead of victim-focused |
| Accountability | Blamed contractors and minimized company role |
| Communication | Dismissive statements that insulted stakeholders |
| Presence | Absent during critical moments (yacht race) |
| Values | Put reputation management above human concerns |
| Composure | Became defensive when challenged |

## Case C: Chesley "Sully" Sullenberger - US Airways Flight 1549 (2009)

### The Crisis

On January 15, 2009, US Airways Flight 1549 struck a flock of geese shortly after takeoff from LaGuardia Airport, disabling both engines. Captain Sullenberger had approximately 208 seconds to make life-or-death decisions.

```
┌─────────────────────────────────────────────────────────────────────┐
│              FLIGHT 1549 TIMELINE                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  3:25:51 PM - Takeoff from LaGuardia                                │
│                                                                       │
│  3:27:11 PM - Bird strike; both engines fail                        │
│               "Mayday, mayday, mayday... this is Cactus 1549"       │
│                                                                       │
│  3:27:32 PM - ATC offers return to LaGuardia                        │
│               Sully: "Unable"                                        │
│                                                                       │
│  3:28:05 PM - ATC offers Teterboro Airport                          │
│               Sully: "We can't do it... we're gonna be              │
│               in the Hudson"                                         │
│                                                                       │
│  3:30:43 PM - Water landing in Hudson River                         │
│                                                                       │
│  3:31:00+ PM - Evacuation begins                                    │
│                                                                       │
│  RESULT: All 155 passengers and crew survived                       │
│  "Miracle on the Hudson"                                             │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Crisis Leadership in Action

```
┌─────────────────────────────────────────────────────────────────────┐
│              SULLENBERGER'S CRISIS LEADERSHIP                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  1. COMPOSURE UNDER EXTREME PRESSURE                                 │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Voice remained calm throughout emergency               │   │
│     │  • Made clear, decisive communications with ATC          │   │
│     │  • Maintained control despite unprecedented situation    │   │
│     │  • Applied decades of training and experience            │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  2. RAPID DECISION-MAKING WITH INCOMPLETE INFORMATION                │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Assessed situation in seconds                         │   │
│     │  • Rejected "safe" options (return to LaGuardia)        │   │
│     │  • Made unconventional decision (water landing)         │   │
│     │  • Committed fully once decision was made               │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  3. TEAM COORDINATION                                                │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Clear communication with First Officer Skiles         │   │
│     │  • Prepared cabin crew for emergency                     │   │
│     │  • Maintained cockpit resource management                │   │
│     │  • Everyone knew their role                              │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  4. RESPONSIBILITY UNTIL THE END                                     │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Walked the cabin twice after landing                  │   │
│     │  • Was last person to leave the aircraft                 │   │
│     │  • Ensured all passengers were accounted for            │   │
│     │  • Put others' safety above his own                      │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  SULLENBERGER'S REFLECTION:                                          │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  "One way of looking at this might be that for 42 years,   │    │
│  │   I've been making small, regular deposits in this bank of │    │
│  │   experience, education and training. And on January 15,   │    │
│  │   the balance was sufficient so that I could make a very   │    │
│  │   large withdrawal."                                        │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Key Lessons from Sullenberger

| Factor | Lesson |
|--------|--------|
| Preparation | Years of training and experience paid off in 208 seconds |
| Calm Under Fire | Emotional regulation enabled clear thinking |
| Decisive Action | When time is critical, decide and commit |
| Servant Leadership | The leader is last to safety |
| Humility | Credited crew, training, and circumstances rather than claiming personal heroism |

## Crisis Leadership Framework

### The RAPID Response Model

```
┌─────────────────────────────────────────────────────────────────────┐
│              THE RAPID CRISIS RESPONSE MODEL                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  R ─── RECOGNIZE                                                     │
│        ┌─────────────────────────────────────────────────────────┐  │
│        │  • Accept reality of the crisis immediately            │  │
│        │  • Don't minimize or deny                              │  │
│        │  • Activate crisis protocols                           │  │
│        └─────────────────────────────────────────────────────────┘  │
│                                                                       │
│  A ─── ASSESS                                                        │
│        ┌─────────────────────────────────────────────────────────┐  │
│        │  • Gather critical information rapidly                 │  │
│        │  • Identify stakeholders affected                      │  │
│        │  • Determine resource needs                            │  │
│        │  • Evaluate options                                    │  │
│        └─────────────────────────────────────────────────────────┘  │
│                                                                       │
│  P ─── PRIORITIZE                                                    │
│        ┌─────────────────────────────────────────────────────────┐  │
│        │  • Human safety first, always                          │  │
│        │  • Then stakeholder impact                             │  │
│        │  • Then operational continuity                         │  │
│        │  • Reputation concerns come last                       │  │
│        └─────────────────────────────────────────────────────────┘  │
│                                                                       │
│  I ─── IMPLEMENT                                                     │
│        ┌─────────────────────────────────────────────────────────┐  │
│        │  • Take decisive action                                │  │
│        │  • Communicate clearly and frequently                  │  │
│        │  • Mobilize resources                                  │  │
│        │  • Adjust as new information emerges                   │  │
│        └─────────────────────────────────────────────────────────┘  │
│                                                                       │
│  D ─── DEBRIEF                                                       │
│        ┌─────────────────────────────────────────────────────────┐  │
│        │  • Learn from the experience                           │  │
│        │  • Document lessons                                    │  │
│        │  • Update protocols                                    │  │
│        │  • Build organizational resilience                     │  │
│        └─────────────────────────────────────────────────────────┘  │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

## Discussion Questions

### Comparative Analysis

1. How did the J&J Credo serve as a decision-making framework during the Tylenol crisis? How can organizations prepare similar guiding principles before crises occur?

2. Why do you think Tony Hayward made the communication errors he did? What pressures might have led to his responses?

3. Sullenberger credited preparation over heroism for the successful outcome. How can organizations build similar "deposits in the bank" for their leaders?

### Application

4. If you were advising a CEO facing a product safety crisis today, what specific communication principles from these cases would you recommend?

5. How has social media changed crisis leadership since the Tylenol case in 1982? What new challenges and opportunities does it create?

6. Develop a personal crisis leadership preparation plan: What can you do now to be ready if you face a crisis?
