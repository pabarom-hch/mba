# Lesson 5: Vision & Communication

## Module 21: Leadership | Estimated Time: 2 Hours

---

## Learning Objectives

By the end of this lesson, you will be able to:

1. **Craft** a compelling organizational vision that inspires action
2. **Apply** storytelling techniques to communicate leadership messages
3. **Develop** executive communication skills for various audiences
4. **Design** communication strategies that cascade through organizations
5. **Evaluate** the effectiveness of leadership communications

---

## Introduction

Vision without communication is hallucination. The most brilliant strategic vision is worthless if leaders cannot articulate it in ways that inspire people to act. This lesson explores how leaders create compelling visions and communicate them effectively to move organizations forward.

## 5. Difficult Conversations

### Framework for Tough Messages

```
┌─────────────────────────────────────────────────────────────────────┐
│              DELIVERING DIFFICULT MESSAGES                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  THE "SUGAR, MEDICINE, SUGAR" APPROACH IS WRONG                     │
│  ────────────────────────────────────────────────                   │
│                                                                       │
│     ✗ AVOID: Positive → Negative → Positive                         │
│       ("Feedback sandwich" undermines credibility)                   │
│                                                                       │
│                                                                       │
│  THE DIRECT BUT EMPATHETIC APPROACH                                  │
│  ────────────────────────────────────                                │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  1. STATE THE FACTS                                         │    │
│  │     "The data shows..." / "Here's the situation..."        │    │
│  │                                                              │    │
│  │  2. ACKNOWLEDGE IMPACT                                       │    │
│  │     "I understand this is difficult..." /                   │    │
│  │     "This will affect..."                                   │    │
│  │                                                              │    │
│  │  3. EXPLAIN THE RATIONALE                                    │    │
│  │     "The reason for this decision is..."                   │    │
│  │                                                              │    │
│  │  4. PROVIDE SUPPORT/NEXT STEPS                               │    │
│  │     "Here's what happens next..." /                         │    │
│  │     "Here's how we'll support you..."                       │    │
│  │                                                              │    │
│  │  5. ALLOW FOR RESPONSE                                       │    │
│  │     "What questions do you have?" /                         │    │
│  │     "Take time to process this"                             │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  COMMON DIFFICULT MESSAGE SCENARIOS:                                 │
│                                                                       │
│  • Layoffs or restructuring                                          │
│  • Project cancellation                                              │
│  • Performance concerns                                              │
│  • Strategic pivot                                                   │
│  • Competitor success                                                │
│  • Crisis communication                                              │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

## Practice Exercises

### Exercise 1: Vision Statement Development

Create a vision statement for your team or organization:

1. Define the current state (where you are)
2. Define the desired future state (where you want to be)
3. Identify the core values that guide you
4. Write a 1-2 sentence vision statement
5. Test it against the IMAGES criteria

### Exercise 2: Story Construction

Develop a leadership story using the framework:

1. Choose a story type (Who I Am, Where We're Going, etc.)
2. Write the hook (30 seconds max)
3. Set the context (who, what, where, when)
4. Describe the challenge (stakes, obstacles)
5. Share the resolution (actions, outcomes)
6. End with a clear call to action
