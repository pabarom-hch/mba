## Learning Objectives

By the end of this lesson, you will be able to:

1. **Define** emotional intelligence and its components using Goleman's framework
2. **Assess** your own emotional intelligence strengths and development areas
3. **Apply** self-awareness techniques to improve leadership effectiveness
4. **Demonstrate** self-regulation strategies in challenging situations
5. **Develop** empathy and social skills for better relationship management

## Introduction

Emotional Intelligence (EQ) refers to the ability to recognize, understand, manage, and effectively use emotions in oneself and others. Research shows that EQ accounts for nearly 90% of what distinguishes star performers from average ones in leadership positions.

### Three Types of Self-Awareness

```
┌─────────────────────────────────────────────────────────────────────┐
│                 THREE TYPES OF SELF-AWARENESS                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  1. EMOTIONAL AWARENESS                                              │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  Recognizing emotions as they occur                        │   │
│     │                                                            │   │
│     │  Questions to ask:                                         │   │
│     │  • What am I feeling right now?                            │   │
│     │  • What triggered this emotion?                            │   │
│     │  • How intense is this feeling (1-10)?                     │   │
│     │  • Is this feeling helping or hindering me?                │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  2. ACCURATE SELF-ASSESSMENT                                         │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  Knowing your strengths and limitations                    │   │
│     │                                                            │   │
│     │  Questions to ask:                                         │   │
│     │  • What do I do exceptionally well?                        │   │
│     │  • Where do I consistently struggle?                       │   │
│     │  • What feedback patterns do I receive?                    │   │
│     │  • Am I over- or under-estimating myself?                  │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  3. SELF-CONFIDENCE                                                  │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  A strong sense of one's self-worth and capabilities       │   │
│     │                                                            │   │
│     │  Signs of healthy self-confidence:                         │   │
│     │  • Present views with conviction                           │   │
│     │  • Make tough decisions despite uncertainty                │   │
│     │  • Acknowledge mistakes without defensiveness              │   │
│     │  • Comfortable being visible and standing out              │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Developing Self-Awareness

| Practice | Description | Frequency |
|----------|-------------|-----------|
| Emotion Journaling | Record emotions, triggers, and responses | Daily |
| Mindfulness Meditation | Non-judgmental awareness of present moment | 10-20 min daily |
| 360-Degree Feedback | Gather perspectives from all directions | Annually |
| Reflection Questions | End-of-day self-assessment | Daily |
| Personality Assessments | MBTI, DiSC, StrengthsFinder | Periodically |
| Coaching | External perspective and accountability | Ongoing |

### The Self-Regulation Process

```
┌─────────────────────────────────────────────────────────────────────┐
│              THE SELF-REGULATION CYCLE                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│        ┌─────────┐                                                   │
│        │ TRIGGER │ ◄─────── External event or internal thought      │
│        └────┬────┘                                                   │
│             │                                                         │
│             ▼                                                         │
│        ┌─────────┐                                                   │

│             │                                                         │
│             ▼                                                         │
│    ┌────────────────┐                                                │
│    │   AWARENESS    │ ◄───── PAUSE: Recognize the emotion           │
│    │    (PAUSE)     │        "I'm feeling frustrated"                │
│    └────────┬───────┘                                                │
│             │                                                         │
│             ▼                                                         │
│    ┌────────────────┐                                                │

│    │   (OPTIONS)    │        "I could react or respond"              │
│    └────────┬───────┘                                                │
│             │                                                         │
│             ▼                                                         │
│    ┌────────────────┐                                                │
│    │   RESPONSE     │ ◄───── ACT: Implement chosen response          │
│    │   (ACTION)     │        "I'll take a breath and ask questions" │
│    └────────┬───────┘                                                │
│             │                                                         │
│             ▼                                                         │
│    ┌────────────────┐                                                │
│    │   OUTCOME      │ ◄───── REFLECT: Evaluate effectiveness        │
│    │  (LEARNING)    │        "That worked better"                    │
│    └────────────────┘                                                │
│                                                                       │
│  KEY INSIGHT: Between stimulus and response lies our power to       │
│  choose. Self-regulation expands that space.                         │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Self-Regulation Techniques

```
┌─────────────────────────────────────────────────────────────────────┐
│              SELF-REGULATION TECHNIQUES                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  IMMEDIATE (In-the-moment)                                           │
│  ─────────────────────────                                           │
│                                                                       │
│  1. PAUSE AND BREATHE                                                │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • 4-7-8 breathing: Inhale 4 sec, hold 7, exhale 8       │   │
│     │  • Box breathing: 4 counts each - in, hold, out, hold    │   │
│     │  • Creates space between stimulus and response            │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  2. REFRAME THE SITUATION                                            │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Ask: "What else could this mean?"                      │   │
│     │  • Consider: "What would I advise a friend?"             │   │
│     │  • Think: "Will this matter in 5 years?"                 │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  3. NAME THE EMOTION                                                 │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • "I'm feeling anxious" (reduces amygdala activity)     │   │
│     │  • Use specific emotion words (frustrated vs. angry)     │   │
│     │  • Engage the rational brain                             │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  ONGOING (Building capacity)                                         │
│  ──────────────────────────                                          │
│                                                                       │
│  4. REGULAR EXERCISE                                                 │
│     └── Reduces baseline stress, improves emotion regulation         │
│                                                                       │
│  5. ADEQUATE SLEEP                                                   │
│     └── Sleep deprivation significantly impairs self-regulation      │
│                                                                       │
│  6. MINDFULNESS PRACTICE                                             │
│     └── Strengthens prefrontal cortex, builds awareness              │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Optimism as a Leadership Skill

| Pessimistic Explanatory Style | Optimistic Explanatory Style |
|------------------------------|------------------------------|
| Permanent: "This always happens" | Temporary: "This happened this time" |

## 5. Empathy

### Three Types of Empathy

```
┌─────────────────────────────────────────────────────────────────────┐
│                    THREE TYPES OF EMPATHY                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                    COGNITIVE EMPATHY                         │    │
│  │                    "I understand you"                        │    │
│  │                                                              │    │
│  │  • Understanding another's perspective                       │    │
│  │  • "Head" empathy - thinking about feelings                 │    │
│  │  • Useful for: negotiations, presentations, feedback        │    │
│  │  • Risk: Can be used manipulatively without other types    │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              │                                       │
│                              ▼                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                    EMOTIONAL EMPATHY                         │    │
│  │                    "I feel with you"                         │    │
│  │                                                              │    │
│  │  • Feeling what another person feels                        │    │
│  │  • "Heart" empathy - experiencing emotions                  │    │
│  │  • Useful for: building trust, coaching, supporting        │    │
│  │  • Risk: Can lead to burnout if not managed                │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                              │                                       │
│                              ▼                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                   COMPASSIONATE EMPATHY                      │    │
│  │                   "I want to help you"                       │    │
│  │                                                              │    │
│  │  • Understanding + feeling + moved to action                │    │
│  │  • Balanced empathy - thinking, feeling, doing              │    │
│  │  • Useful for: leadership, service, problem-solving        │    │
│  │  • Ideal: Combines understanding with appropriate action   │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  Effective leaders use all three types appropriately:               │
│  • Cognitive empathy to understand stakeholder perspectives         │
│  • Emotional empathy to connect authentically                       │
│  • Compassionate empathy to take meaningful action                  │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

## Practice Exercises

### Exercise 1: Emotion Tracking

For one week, track your emotions using this template:

| Time | Situation | Emotion | Intensity (1-10) | Response | Effectiveness |
|------|-----------|---------|------------------|----------|---------------|
| | | | | | |

### Exercise 2: Empathy Practice

In your next three conversations:
1. Focus entirely on understanding the other person's perspective
2. Reflect back what you hear before responding
3. Ask at least two clarifying questions
4. Notice what you learn that you might have missed otherwise

### Exercise 3: Self-Regulation Challenge

Choose one trigger situation and practice:
1. Identify the typical trigger
2. Notice your automatic emotional response
3. Use the pause-and-breathe technique
4. Choose a different response
5. Reflect on the outcome
