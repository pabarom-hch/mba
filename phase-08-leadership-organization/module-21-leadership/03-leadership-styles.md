# Lesson 3: Leadership Styles

## Module 21: Leadership | Estimated Time: 2 Hours

---

## Learning Objectives

By the end of this lesson, you will be able to:

1. **Identify** Goleman's six leadership styles and their characteristics
2. **Recognize** when to apply each leadership style effectively
3. **Analyze** the impact of different styles on organizational climate
4. **Develop** flexibility to switch between styles based on context
5. **Create** a personal leadership style portfolio

---

## Introduction

Effective leaders don't rely on a single leadership style. Like a golfer selecting the right club for each shot, skilled leaders choose from a repertoire of styles based on the situation, the people involved, and the outcomes needed. Daniel Goleman's research identified six distinct leadership styles, each creating a different organizational climate.

## 1. Visionary (Authoritative) Style

### Characteristics

```
┌─────────────────────────────────────────────────────────────────────┐
│                    VISIONARY LEADERSHIP STYLE                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  MOTTO: "Come with me"                                               │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                    LEADER APPROACH                           │    │
│  │                                                              │    │
│  │  • Articulates a compelling vision of the future            │    │
│  │  • Explains how work connects to larger purpose             │    │
│  │  • Gives freedom in how goals are achieved                  │    │
│  │  • Provides clear direction, not micromanagement            │    │
│  │  • Inspires commitment through meaning                      │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  EQ COMPETENCIES:                  CLIMATE IMPACT: +54               │
│  • Self-confidence                 (Highest positive impact)         │
│  • Empathy                                                           │
│  • Change catalyst                                                   │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                     WHEN TO USE                              │    │
│  │                                                              │    │
│  │  ✓ New direction needed                                     │    │
│  │  ✓ Organizational change or transformation                  │    │
│  │  ✓ Clear standards needed for success                       │    │
│  │  ✓ Team needs to understand the "why"                       │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                     WHEN TO AVOID                            │    │
│  │                                                              │    │
│  │  ✗ Working with experts who know more than you              │    │
│  │  ✗ Team needs detailed tactical guidance                    │    │
│  │  ✗ Leader lacks credibility or trust                        │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  EXAMPLE: Steve Jobs returning to Apple and articulating the         │
│  "Think Different" vision that transformed the company               │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │

## Practice Exercises

### Exercise 1: Style Identification

Read each scenario and identify which leadership style would be most effective:

1. Your team just received devastating news about a major project cancellation. Morale is at an all-time low.

2. A new team member has high potential but lacks experience. She's eager to learn but makes frequent mistakes.

3. You've just been promoted to lead a transformation initiative that will change how the entire division operates.

4. Your highly skilled engineering team needs to solve a complex technical problem and you're not sure of the best approach.

5. A fire drill alarm goes off during a crowded company event.

### Exercise 2: Personal Development Plan

Based on your style assessment:

1. Identify your two default styles
2. Identify two styles you underuse
3. For each underused style, identify one specific situation where it would help
4. Create a 30-day practice plan for developing one new style

### Exercise 3: Style Switching Simulation

Think of a recent leadership situation. Write out how you would handle it using each of the six styles. Consider:
- What would you say differently?
- How would your body language change?
- What outcomes might be different?
