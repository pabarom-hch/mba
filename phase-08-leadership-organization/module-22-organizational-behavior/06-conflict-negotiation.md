By the end of this lesson, you will be able to:

1. **Identify** different types and sources of organizational conflict

3. **Conduct** interest-based negotiations using the Harvard method
4. **Transform** conflict from destructive to constructive
5. **Facilitate** resolution of disputes between team members

## Introduction

Conflict is inevitable in organizations. Different perspectives, competing resources, and diverse interests create friction. The goal is not to eliminate conflict but to manage it constructively. When handled well, conflict leads to better decisions, stronger relationships, and innovation. When handled poorly, it destroys teams, wastes resources, and drives away talent.

### Sources of Conflict

| Source | Description | Example |
|--------|-------------|---------|
| Scarce Resources | Competition for limited budgets, people, equipment | Two departments want the same budget |
| Goal Differences | Incompatible objectives | Sales wants volume; Finance wants margin |
| Interdependence | Reliance on others who have different priorities | Manufacturing depends on R&D timelines |
| Ambiguity | Unclear roles, responsibilities, or authority | Who owns the customer relationship? |
| Communication | Misunderstandings, poor information flow | Email misinterpreted as hostile |
| Values | Different beliefs about what matters | Work-life balance vs. 24/7 availability |

## 2. Conflict Management Styles

### Thomas-Kilmann Conflict Mode Instrument

```
┌─────────────────────────────────────────────────────────────────────┐
│              CONFLICT MANAGEMENT STYLES                              │
│              (Thomas-Kilmann)                                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  ASSERTIVENESS (Satisfying own concerns)                             │
│      │                                                               │
│  HIGH├───────────────────────┬───────────────────────┐              │
│      │                       │                       │              │
│      │      COMPETING        │     COLLABORATING     │              │
│      │                       │                       │              │
│      │   "My way or the      │   "Let's find a      │              │
│      │    highway"           │    win-win"          │              │
│      │                       │                       │              │
│      │   Win-Lose            │   Win-Win            │              │
│      │                       │                       │              │
│      ├───────────────────────┼───────────────────────┤              │
│      │                       │                       │              │
│      │                    COMPROMISING              │              │
│      │                       │                       │              │
│      │                 "Let's split the             │              │
│      │                  difference"                 │              │
│      │                       │                       │              │
│      │                 Partial Win-Win             │              │
│      │                       │                       │              │
│      ├───────────────────────┼───────────────────────┤              │
│      │                       │                       │              │
│      │      AVOIDING         │    ACCOMMODATING      │              │
│      │                       │                       │              │
│      │   "I'll deal with     │   "Whatever you      │              │
│      │    it later"          │    want"             │              │
│      │                       │                       │              │
│      │   Lose-Lose           │   Lose-Win           │              │
│      │                       │                       │              │
│  LOW └───────────────────────┴───────────────────────┘              │
│              LOW        COOPERATIVENESS          HIGH                │
│                    (Satisfying others' concerns)                     │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### When to Use Each Style

| Style | When to Use | When to Avoid |
|-------|-------------|---------------|
| **Competing** | Emergency; you know you're right; others are taking advantage | Relationships matter; you need buy-in; you might be wrong |
| **Collaborating** | Complex issues; time available; need commitment; relationship matters | Trivial issues; urgent decisions; others don't care |
| **Compromising** | Moderate importance; time pressure; temporary solution needed | Problem can be solved better with more time |
| **Avoiding** | Trivial issue; need to cool down; more info needed | Important issue; problem will get worse |
| **Accommodating** | You're wrong; relationship more important than issue; goodwill | Enables exploitation; principle at stake |

### Harvard Negotiation Method

```
┌─────────────────────────────────────────────────────────────────────┐
│              PRINCIPLED NEGOTIATION                                  │
│              (Fisher & Ury - "Getting to Yes")                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  FOUR PRINCIPLES:                                                    │
│                                                                       │
│  1. SEPARATE PEOPLE FROM THE PROBLEM                                 │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Be hard on the problem, soft on the people            │   │
│     │  • Attack issues, not individuals                        │   │
│     │  • Build relationship while addressing substance         │   │
│     │                                                           │   │
│     │  Instead of: "You're being unreasonable"                 │   │
│     │  Say: "I'm having trouble with this approach"           │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  2. FOCUS ON INTERESTS, NOT POSITIONS                                │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Ask "Why?" and "Why not?"                             │   │
│     │  • Look for underlying motivations                       │   │
│     │  • Identify shared and compatible interests              │   │
│     │                                                           │   │
│     │  Ask: "Help me understand why this matters to you"       │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  3. GENERATE OPTIONS FOR MUTUAL GAIN                                 │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Brainstorm before deciding                            │   │
│     │  • Expand the pie before dividing it                     │   │
│     │  • Look for creative solutions                           │   │
│     │                                                           │   │
│     │  Say: "What if we tried..." / "Another option might be..."│  │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  4. USE OBJECTIVE CRITERIA                                           │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  • Base decisions on standards, not power                │   │
│     │  • Use benchmarks, precedent, market value               │   │
│     │  • Be open to reason                                     │   │
│     │                                                           │   │
│     │  Ask: "What standard should we use to decide this?"      │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  BATNA: BEST ALTERNATIVE TO NEGOTIATED AGREEMENT                     │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  • Know your walkaway option before negotiating             │    │
│  │  • The better your BATNA, the more power you have          │    │
│  │  • Never accept less than your BATNA                       │    │
│  │  • Try to improve your BATNA before negotiating            │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

## 4. Conflict Resolution Process

### The Resolution Framework

```
┌─────────────────────────────────────────────────────────────────────┐
│              CONFLICT RESOLUTION PROCESS                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  STEP 1: CREATE THE RIGHT ENVIRONMENT                                │
│  ─────────────────────────────────────                               │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Choose neutral, private location                        │      │
│  │  • Ensure adequate time (don't rush)                      │      │
│  │  • Set ground rules (no interrupting, respect)            │      │
│  │  • Start with shared goal (resolving the issue)           │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  STEP 2: UNDERSTAND ALL PERSPECTIVES                                 │
│  ───────────────────────────────────                                 │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Let each person tell their story uninterrupted         │      │
│  │  • Listen actively and empathetically                     │      │
│  │  • Ask clarifying questions                               │      │
│  │  • Summarize to show understanding                        │      │
│  │  • Identify underlying interests                          │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  STEP 3: FIND COMMON GROUND                                          │
│  ──────────────────────────                                          │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Identify shared goals and interests                    │      │
│  │  • Acknowledge areas of agreement                         │      │
│  │  • Reframe as "us vs. the problem"                       │      │
│  │  • Build on areas of common interest                     │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  STEP 4: GENERATE OPTIONS                                            │
│  ────────────────────────                                            │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Brainstorm without judging                             │      │
│  │  • Encourage creative solutions                           │      │
│  │  • Consider win-win possibilities                         │      │
│  │  • Expand options before narrowing                        │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  STEP 5: REACH AGREEMENT                                             │
│  ───────────────────────                                             │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Evaluate options against interests                     │      │
│  │  • Use objective criteria                                 │      │
│  │  • Confirm understanding and commitment                   │      │
│  │  • Document agreement (who does what by when)             │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  STEP 6: FOLLOW UP                                                   │
│  ─────────────                                                       │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Check in on implementation                             │      │
│  │  • Address any new issues quickly                         │      │
│  │  • Celebrate successful resolution                        │      │
│  │  • Learn from the process                                 │      │
│  └───────────────────────────────────────────────────────────┘      │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Difficult Conversation Tips

| Challenge | Technique |
|-----------|-----------|
| Emotions running high | Acknowledge feelings first: "I can see this is frustrating" |
| Person attacks you | Don't defend; redirect: "I want to understand your concern" |
| Stuck on positions | Ask "Why is that important to you?" |
| One person dominating | Redirect: "I want to make sure I hear from everyone" |
| Deadlock | Take a break; introduce new options; bring in third party |

## 5. Building Conflict Competence

### For Managers: Facilitating Conflict Resolution

```
┌─────────────────────────────────────────────────────────────────────┐
│              MANAGER'S ROLE IN CONFLICT                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  WHEN TO INTERVENE:                                                  │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  YES - Intervene when:                                       │    │
│  │  • Work is being affected                                   │    │
│  │  • Conflict is escalating                                   │    │
│  │  • Someone requests help                                    │    │
│  │  • Issues involve harassment or discrimination             │    │
│  │  • Parties have tried and failed to resolve               │    │
│  │                                                              │    │
│  │  NO - Let them work it out when:                            │    │
│  │  • Issue is minor                                           │    │
│  │  • Parties are capable of resolving                        │    │
│  │  • Intervention would undermine autonomy                   │    │
│  │  • Learning opportunity exists                             │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  HOW TO FACILITATE:                                                  │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  • Stay neutral (don't take sides)                         │    │
│  │  • Meet with parties separately first if needed            │    │
│  │  • Focus on behaviors, not personalities                   │    │
│  │  • Keep conversation forward-focused                       │    │
│  │  • Don't solve it for them (facilitate, don't dictate)    │    │
│  │  • Document agreements                                      │    │
│  │  • Follow up to ensure resolution holds                    │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  QUESTIONS TO ASK:                                                   │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  • "Help me understand what happened"                       │    │
│  │  • "What's your ideal outcome here?"                       │    │
│  │  • "What do you think [other person] needs?"              │    │
│  │  • "What would you be willing to do differently?"         │    │
│  │  • "What can we all agree on?"                            │    │
│  │  • "How can I help?"                                       │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```
