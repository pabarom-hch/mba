## Learning Objectives

By the end of this lesson, you will be able to:

1. **Compare** major motivation theories and their workplace applications
2. **Assess** how personality differences affect job performance and fit
3. **Recognize** perceptual biases that influence workplace judgments
4. **Apply** motivation frameworks to improve employee engagement
5. **Design** jobs that enhance intrinsic motivation

## Introduction

Understanding individual behavior is the foundation of organizational behavior. Why do some employees go above and beyond while others do the minimum? Why do people with similar skills perform so differently? This lesson explores the psychological factors that drive individual behavior at work.

### Process Theories: How Motivation Works

#### Expectancy Theory (Vroom)

```
┌─────────────────────────────────────────────────────────────────────┐
│              EXPECTANCY THEORY                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  MOTIVATION = EXPECTANCY × INSTRUMENTALITY × VALENCE                 │
│                                                                       │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐           │
│  │  EXPECTANCY  │ ×  │INSTRUMENTA-  │ ×  │   VALENCE    │           │
│  │              │    │    LITY      │    │              │           │
│  │   E → P      │    │   P → O      │    │   V(O)       │           │
│  │              │    │              │    │              │           │
│  │ "If I try,   │    │ "If I       │    │ "Do I want   │           │
│  │  can I       │    │  perform,    │    │  those       │           │
│  │  perform?"   │    │  will I get  │    │  outcomes?"  │           │
│  │              │    │  outcomes?"  │    │              │           │
│  └──────┬───────┘    └──────┬───────┘    └──────┬───────┘           │
│         │                   │                   │                    │
│         ▼                   ▼                   ▼                    │
│    0.0 ──────── 1.0   0.0 ──────── 1.0   -1.0 ──── 0 ──── +1.0     │
│                                                                       │
│  EXAMPLE CALCULATION:                                                │
│                                                                       │
│  Sales rep considering extra effort:                                 │
│  • Expectancy: 0.8 (confident effort → sales)                       │
│  • Instrumentality: 0.6 (sales → bonus)                             │
│  • Valence: +0.9 (bonus is highly valued)                           │
│                                                                       │
│  Motivation = 0.8 × 0.6 × 0.9 = 0.43                                │
│                                                                       │
│  MANAGER IMPLICATIONS:                                               │
│                                                                       │
│  To increase Expectancy:    │ Training, confidence-building,        │
│                             │ clear goals, resources                 │
│  To increase Instrumentality:│ Clear performance criteria,          │
│                             │ consistent rewards, credibility       │
│  To increase Valence:       │ Understand what employees value,      │
│                             │ offer meaningful rewards               │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Common Perceptual Biases

```
┌─────────────────────────────────────────────────────────────────────┐
│              COMMON PERCEPTUAL BIASES                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  HALO EFFECT                                                         │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  Drawing general impression from single characteristic       │    │
│  │  Example: "She's articulate, so she must be smart,          │    │
│  │           organized, and a good leader."                     │    │
│  │  Impact: Hiring, performance reviews                        │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  CONFIRMATION BIAS                                                   │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  Seeking information that confirms existing beliefs          │    │
│  │  Example: Manager who thinks employee is lazy notices only  │    │
│  │           instances of tardiness, not extra effort          │    │
│  │  Impact: Performance management, decision making            │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  RECENCY EFFECT                                                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  Recent events weighted more heavily than earlier ones       │    │
│  │  Example: Annual review based mainly on last month          │    │
│  │  Impact: Performance evaluations                            │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  FUNDAMENTAL ATTRIBUTION ERROR                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  Overattributing others' behavior to personality vs.        │    │
│  │  situation                                                   │    │
│  │  Example: "He's late because he's lazy" (ignoring traffic) │    │
│  │  Impact: Performance feedback, team dynamics                │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  SELF-SERVING BIAS                                                   │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  Attributing success to internal factors, failure to        │    │
│  │  external factors                                            │    │
│  │  Example: "I succeeded because of my skill; I failed       │    │
│  │           because of bad luck."                             │    │
│  │  Impact: Learning from mistakes, accountability             │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  SIMILAR-TO-ME EFFECT                                                │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  Favoring people who are similar to ourselves                │    │
│  │  Example: Hiring candidates with same background/interests  │    │
│  │  Impact: Diversity, hiring decisions                        │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Attribution Theory

```
┌─────────────────────────────────────────────────────────────────────┐
│              ATTRIBUTION THEORY (Kelley)                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  How do we determine if behavior is internally or externally caused?│
│                                                                       │
│  THREE FACTORS:                                                      │
│                                                                       │
│  ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐      │
│  │   DISTINCTIVENESS   │   CONSENSUS    │   │  CONSISTENCY    │      │
│  └─────────┬───────┘  └────────┬────────┘  └────────┬────────┘      │
│            │                   │                    │                │
│  Does person           Do others             Does person            │
│  behave this way       behave the            always behave          │
│  in other              same way in           this way in            │
│  situations?           this situation?       this situation?        │
│                                                                       │
│  ATTRIBUTION PATTERNS:                                               │
│                                                                       │
│  INTERNAL ATTRIBUTION (personality, ability, effort)                 │
│  ├── Low distinctiveness (acts this way in many situations)         │
│  ├── Low consensus (others don't act this way)                      │
│  └── High consistency (always acts this way here)                   │
│                                                                       │
│  EXTERNAL ATTRIBUTION (situation, luck, difficulty)                  │
│  ├── High distinctiveness (acts differently elsewhere)              │
│  ├── High consensus (others act the same way)                       │
│  └── High consistency (always acts this way here)                   │
│                                                                       │
│  EXAMPLE:                                                            │
│  Employee misses deadline                                            │
│  • If she usually meets deadlines (high distinctiveness)            │
│  • If others also struggled (high consensus)                        │
│  • If this project always runs late (high consistency)              │
│  → EXTERNAL attribution (difficult project)                         │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

## 4. Job Design for Motivation

### Job Characteristics Model (Hackman & Oldham)

```
┌─────────────────────────────────────────────────────────────────────┐
│              JOB CHARACTERISTICS MODEL                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  CORE JOB            CRITICAL              PERSONAL & WORK          │
│  CHARACTERISTICS     PSYCHOLOGICAL STATES   OUTCOMES                 │
│  ───────────────     ──────────────────     ─────────────────       │
│                                                                       │
│  ┌─────────────────┐                                                │
│  │ Skill Variety   │                       ┌────────────────┐       │
│  └────────┬────────┘                       │                │       │
│           │         ┌─────────────────┐    │  • High        │       │
│  ┌────────┴────────┐│ Experienced     │    │    internal    │       │
│  │ Task Identity   │┤ Meaningfulness  ├────│    work        │       │
│  └────────┬────────┘│ of work         │    │    motivation  │       │
│           │         └─────────────────┘    │                │       │
│  ┌────────┴────────┐                       │  • High-       │       │
│  │ Task            │                       │    quality     │       │
│  │ Significance    │                       │    work        │       │
│  └─────────────────┘                       │    performance │       │
│                                            │                │       │
│  ┌─────────────────┐┌─────────────────┐    │  • High        │       │
│  │ Autonomy        ││ Experienced     │    │    satisfaction│       │
│  │                 ├┤ Responsibility  ├────│    with work   │       │
│  └─────────────────┘│ for outcomes    │    │                │       │
│                     └─────────────────┘    │  • Low         │       │
│                                            │    absenteeism │       │
│  ┌─────────────────┐┌─────────────────┐    │    & turnover  │       │
│  │ Feedback        ││ Knowledge of    │    │                │       │
│  │                 ├┤ actual results  ├────┤                │       │
│  └─────────────────┘│                 │    └────────────────┘       │
│                     └─────────────────┘                              │
│                                                                       │
│  MOTIVATING POTENTIAL SCORE (MPS):                                   │
│                                                                       │
│  MPS = [(Skill Variety + Task Identity + Task Significance) / 3]    │
│        × Autonomy × Feedback                                         │
│                                                                       │
│  MODERATOR: Growth Need Strength                                     │
│  (Some employees value growth more than others)                      │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Job Design Strategies

| Strategy | Description | Example |
|----------|-------------|---------|
| Job Rotation | Moving employees between jobs | Assembly line worker rotating stations |
| Job Enlargement | Adding more tasks at same level | Receptionist also handling mail |
| Job Enrichment | Adding higher-level responsibilities | Customer service rep handling complaints |
| Empowerment | Giving decision-making authority | Team self-scheduling |
| Flexible Work | Time/location flexibility | Remote work options |

## Practice Exercises

### Exercise 1: Motivation Diagnosis

For an unmotivated employee, use Expectancy Theory to diagnose the issue:

1. Does the employee believe effort leads to performance? (Expectancy)
2. Does the employee believe performance leads to rewards? (Instrumentality)
3. Does the employee value the available rewards? (Valence)

What interventions would address each gap?

### Exercise 2: Job Redesign

Analyze a job you know well using the Job Characteristics Model:

1. Rate each core characteristic (1-7)
2. Calculate the Motivating Potential Score
3. Identify which characteristics are lowest
4. Propose specific changes to enhance motivation

### Exercise 3: Bias Awareness

Over the next week, notice your perceptions of others:

1. Catch yourself making quick judgments
2. Identify which bias might be at play
3. Challenge your initial perception
4. Seek additional information
