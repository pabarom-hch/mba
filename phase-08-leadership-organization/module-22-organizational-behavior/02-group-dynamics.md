# Lesson 2: Group Dynamics

## Module 22: Organizational Behavior | Estimated Time: 2 Hours

---

## Learning Objectives

By the end of this lesson, you will be able to:

1. **Differentiate** between groups and teams and their characteristics
2. **Apply** Tuckman's stages of group development to team situations
3. **Identify** team roles and their contribution to effectiveness
4. **Recognize** group decision-making pitfalls including groupthink
5. **Design** interventions to improve team dynamics

## Introduction

Humans are inherently social, and work happens increasingly through teams. Understanding group dynamics is essential for managers because groups can either amplify individual contributions or diminish them. This lesson explores how groups form, develop, and function.

```

│                                                                       │
│  STAGE 1: FORMING                                                    │
│  ─────────────────                                                   │
│  Characteristics:                                                    │
│  • Polite, tentative, testing boundaries                            │
│  • Dependence on leader for direction                               │
│  • Uncertainty about purpose and roles                              │
│                                                                       │
│  Leader Actions:                                                     │
│  • Provide clear direction and structure                            │

│                                                                       │
│  STAGE 2: STORMING                                                   │
│  ──────────────────                                                  │
│  Characteristics:                                                    │
│  • Conflict over control and influence                              │

│  • Frustration with slow progress                                   │
│                                                                       │
│  Leader Actions:                                                     │
│  • Acknowledge conflict as normal                                   │
│  • Facilitate resolution, don't suppress                            │

│                                                                       │
│  ═══════════════════════════════════════════════════════════════    │
│                                                                       │
│  STAGE 3: NORMING                                                    │
│  ─────────────────                                                   │

│  • Norms and standards established                                  │
│  • Trust and openness increase                                      │
│  • Collaboration becomes natural                                    │
│                                                                       │
│  Leader Actions:                                                     │
│  • Step back, allow team autonomy                                   │
│  • Reinforce productive norms                                       │
│  • Encourage full participation                                     │
│                                                                       │
│  ═══════════════════════════════════════════════════════════════    │
│                                                                       │
│  STAGE 4: PERFORMING                                                 │
│  ────────────────────                                                │
│  Characteristics:                                                    │
│  • High productivity and effectiveness                              │
│  • Flexibility and interdependence                                  │
│  • Problems solved constructively                                   │
│  • Focus on achieving results                                       │
│                                                                       │
│  Leader Actions:                                                     │
│  • Delegate and empower                                             │
│  • Focus on development and stretch goals                          │
│  • Celebrate successes                                              │

│  ────────────────────                                                │

│  • Recognition and celebration                                      │
│                                                                       │
│  Leader Actions:                                                     │
│  • Acknowledge contributions                                        │
│  • Capture lessons learned                                          │
│  • Facilitate closure and transition                               │
│                                                                       │

```

### Recognizing Team Stages

| Stage | Signs | Questions Team Asks |
|-------|-------|---------------------|
| Forming | Politeness, hesitation, looking to leader | "What are we doing? Who are these people?" |
| Storming | Disagreements, tension, resistance | "Who's in charge? Why should I listen to you?" |
| Norming | Cooperation, cohesion, humor | "How do we work together? What are our rules?" |
| Performing | Productivity, flexibility, interdependence | "How can we achieve our goals better?" |
| Adjourning | Completion, reflection, emotions | "What did we accomplish? What's next?" |

### Preventing Groupthink

| Strategy | Implementation |
|----------|----------------|
| Devil's Advocate | Assign someone to challenge assumptions |
| Outside Experts | Bring in fresh perspectives |
| Anonymous Input | Use written or electronic voting |
| Leader Restraint | Leader speaks last, encourages dissent |
| Subgroups | Break into smaller groups, then compare conclusions |
| Second Chance | Revisit decision after reflection period |

### Team Size Considerations

| Size | Advantages | Disadvantages |
|------|------------|---------------|
| 2-4 | High interaction, fast decisions | Limited perspectives, workload issues |
| 5-7 | Ideal for most tasks | Requires good facilitation |
| 8-12 | Diverse perspectives | Harder coordination, subgroups form |
| 12+ | Many resources | Social loafing, communication breakdown |

## Practice Exercises

### Exercise 1: Stage Diagnosis

Think of a team you're currently on or have been part of:

1. What stage is the team currently in?
2. What specific behaviors indicate this stage?
3. What would help move to the next stage?
4. What might cause regression to an earlier stage?

### Exercise 2: Role Analysis

For your team:

1. List team members and their primary Belbin roles
2. Identify which roles are over-represented
3. Identify which roles are missing
4. Propose adjustments (adding members, role-switching, hiring)

### Exercise 3: Decision Quality Check

For an upcoming team decision:

1. List potential groupthink symptoms to watch for
2. Plan which prevention strategies you'll use
3. Identify who could serve as devil's advocate
4. Design the decision process to maximize quality
