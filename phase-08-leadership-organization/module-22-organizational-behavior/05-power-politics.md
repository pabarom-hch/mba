# Lesson 5: Power & Politics

## Module 22: Organizational Behavior | Estimated Time: 2 Hours

---

## Learning Objectives

By the end of this lesson, you will be able to:

1. **Distinguish** between different sources of power in organizations
2. **Apply** influence tactics appropriate to various situations
3. **Navigate** organizational politics ethically and effectively
4. **Build** political skill as a leadership competency
5. **Recognize** the difference between functional and dysfunctional politics

---

## Introduction

Power and politics are realities of organizational life. While often viewed negatively, they are essential tools for getting things done, especially when formal authority is insufficient. Understanding power dynamics and developing political skill enables leaders to advance initiatives, build coalitions, and create positive change.

## 1. Sources of Power

### French and Raven's Power Bases

```
┌─────────────────────────────────────────────────────────────────────┐
│              SOURCES OF POWER (French & Raven)                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  POSITIONAL POWER (Formal Authority)                                 │
│  ═══════════════════════════════════                                 │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  LEGITIMATE POWER                                            │    │
│  │  Based on: Formal position in hierarchy                     │    │
│  │  Example: Manager can assign work to direct reports         │    │
│  │  Strength: Clear, accepted                                  │    │
│  │  Weakness: Limited to formal scope; can breed resentment   │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  REWARD POWER                                                │    │
│  │  Based on: Ability to provide valued outcomes               │    │
│  │  Example: Manager can give raises, promotions, assignments  │    │
│  │  Strength: Motivating when used fairly                      │    │
│  │  Weakness: Compliance-focused; can feel transactional       │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  COERCIVE POWER                                              │    │
│  │  Based on: Ability to punish or withhold                    │    │
│  │  Example: Manager can fire, demote, assign undesirable work │    │
│  │  Strength: Gains compliance quickly                         │    │
│  │  Weakness: Creates fear, resistance, and turnover          │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  PERSONAL POWER (Earned Influence)                                   │
│  ═════════════════════════════════                                   │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  EXPERT POWER                                                │    │
│  │  Based on: Specialized knowledge and skills                 │    │
│  │  Example: Data scientist's analytical expertise             │    │
│  │  Strength: Highly respected; works across boundaries        │    │
│  │  Weakness: Narrow; must be maintained through learning     │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  REFERENT POWER                                              │    │
│  │  Based on: Personal attraction and identification           │    │
│  │  Example: Admired leader whom others want to emulate        │    │
│  │  Strength: Creates loyalty and commitment                   │    │
│  │  Weakness: Takes time to build; can be lost quickly        │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ADDITIONAL SOURCES                                                  │
│  ═════════════════                                                   │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  INFORMATION POWER: Control over data and knowledge flow    │    │
│  │  CONNECTION POWER: Network and relationships               │    │
│  │  SITUATIONAL POWER: Being in right place at right time     │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Power Effectiveness

| Power Base | Compliance | Commitment | Resistance | Best Use |
|------------|------------|------------|------------|----------|
| Legitimate | Moderate | Low | Low | Routine requests |
| Reward | High | Moderate | Low | Motivating extra effort |
| Coercive | High | None | High | Last resort, safety |
| Expert | High | High | Low | Complex problems |
| Referent | High | Very High | None | Building loyalty |

## 2. Influence Tactics

### The Influence Toolkit

```
┌─────────────────────────────────────────────────────────────────────┐
│              INFLUENCE TACTICS                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  RATIONAL/LOGICAL TACTICS                                            │
│  ─────────────────────────                                           │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  RATIONAL PERSUASION                                         │    │
│  │  Using facts, data, and logical arguments                   │    │
│  │  Example: "The data shows a 20% improvement with this approach"│  │
│  │  Best when: Target values logic; you have strong evidence   │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  CONSULTATION                                                │    │
│  │  Seeking input to gain ownership                            │    │
│  │  Example: "How do you think we should approach this?"       │    │
│  │  Best when: You need buy-in; target has relevant expertise │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  EMOTIONAL/RELATIONAL TACTICS                                        │
│  ────────────────────────────                                        │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  INSPIRATIONAL APPEAL                                        │    │
│  │  Appealing to values, ideals, and aspirations               │    │
│  │  Example: "This aligns with what we all care about"        │    │
│  │  Best when: Target cares about vision; emotional resonance │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  PERSONAL APPEAL                                             │    │
│  │  Asking as a favor based on relationship                    │    │
│  │  Example: "I really need your help on this"                │    │
│  │  Best when: Strong relationship exists; rare asks           │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  INGRATIATION                                                │    │
│  │  Building goodwill before making request                    │    │
│  │  Example: Complimenting, doing favors, being friendly       │    │
│  │  Caution: Can seem manipulative if overdone                │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  STRUCTURAL/INDIRECT TACTICS                                         │
│  ───────────────────────────                                         │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  COALITION BUILDING                                          │    │
│  │  Enlisting others to support your position                  │    │
│  │  Example: "Marketing and Finance both support this approach"│   │
│  │  Best when: You lack authority; need collective voice      │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  EXCHANGE                                                    │    │
│  │  Offering something in return                               │    │
│  │  Example: "If you support this, I'll help with your project"│   │
│  │  Best when: Mutual benefit is clear; relationship allows it │   │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  LEGITIMATING                                                │    │
│  │  Citing rules, policies, or precedent                       │    │
│  │  Example: "The policy requires..."                         │    │
│  │  Best when: Clear authority exists; organizational backing │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  HARD TACTICS (Use Sparingly)                                        │
│  ─────────────────────────────                                       │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  PRESSURE                                                    │    │
│  │  Using demands, threats, or persistent reminders           │    │
│  │  Example: "This needs to happen by Friday"                 │    │
│  │  Caution: Creates compliance but damages relationship      │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Influence by Direction

| Direction | Most Effective Tactics |
|-----------|----------------------|
| Upward (to boss) | Rational persuasion, consultation, inspirational appeal |
| Lateral (to peers) | Consultation, exchange, coalition, personal appeal |
| Downward (to reports) | Rational persuasion, inspirational appeal, consultation |

### Ethical Guidelines

| Principle | Application |
|-----------|-------------|
| Transparency | Be honest about your interests and intentions |
| Win-Win Focus | Seek outcomes that benefit others, not just you |
| Relationship First | Build genuine relationships, not just transactions |
| Long-Term View | Protect your reputation; short-term wins aren't worth it |
| Substance Over Spin | Have genuinely good ideas worth advocating |
| Fairness | Compete on merits; don't undermine others unfairly |

### Exercise 2: Influence Planning

For an initiative you want to advance:

1. Who are the key decision-makers?
2. What are their interests and concerns?
3. What influence tactics would work best with each?
4. Who could be an ally? Who might resist?
5. What's your sequence of engagement?

### Exercise 3: Political Skill Development

Identify one dimension to develop:

1. Social Astuteness: Observe three meetings and note dynamics
2. Interpersonal Influence: Practice one new influence tactic
3. Networking: Have coffee with someone outside your function
4. Apparent Sincerity: Share something genuine about yourself
