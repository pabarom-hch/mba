## Introduction

What makes some teams exceptional while others with similar talent underperform? Research reveals that high-performing teams aren't simply collections of star individuals. They have specific characteristics, norms, and dynamics that create collective intelligence exceeding the sum of individual abilities.

### Dysfunction 3: Lack of Commitment

```
┌─────────────────────────────────────────────────────────────────────┐
│              DYSFUNCTION 3: LACK OF COMMITMENT                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  DEFINITION: Failure to clearly commit to decisions, creating       │
│  ambiguity about direction and priorities                            │
│                                                                       │
│  TWO CAUSES OF LACK OF COMMITMENT:                                   │
│                                                                       │
│  1. DESIRE FOR CONSENSUS                                             │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  Waiting for everyone to agree before moving forward      │   │
│     │  Reality: Reasonable people don't need consensus;         │   │
│     │  they need to be heard.                                   │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  2. NEED FOR CERTAINTY                                               │
│     ┌───────────────────────────────────────────────────────────┐   │
│     │  Waiting for complete information before deciding         │   │
│     │  Reality: Better to make a decision and be wrong than    │   │
│     │  waffle and be unclear.                                   │   │
│     └───────────────────────────────────────────────────────────┘   │
│                                                                       │
│  TEAMS LACKING COMMITMENT:       TEAMS WITH COMMITMENT:              │
│  ──────────────────────────      ─────────────────────               │

│  • Endless discussion           • Alignment around objectives        │
│  • Second-guessing decisions    • Learn from mistakes                │
│  • Lack of confidence           • Change direction quickly           │
│  • Fear of failure              • Move forward without hesitation    │
│                                                                       │
│  BUILDING COMMITMENT:                                                │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  CASCADING MESSAGING                                        │    │
│  │  End meetings by reviewing decisions and agreeing on        │    │
│  │  what to communicate to teams.                              │    │
│  │                                                              │    │
│  │  DEADLINES                                                  │    │
│  │  Set clear deadlines for decisions, not just tasks.        │    │
│  │                                                              │    │
│  │  CONTINGENCY PLANNING                                       │    │
│  │  Identify worst-case scenarios to reduce fear.             │    │
│  │                                                              │    │
│  │  LOW-RISK EXPOSURE                                          │    │
│  │  Make small decisions quickly to build habit.              │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │

## 2. Psychological Safety

### Building Psychological Safety

```

│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  THE FIVE KEYS TO TEAM EFFECTIVENESS:                                │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │

### Implementing the Five Keys

| Key | Assessment Question | Improvement Actions |
|-----|---------------------|---------------------|
| Psychological Safety | "If I make a mistake, it won't be held against me" | Leader models vulnerability; celebrate learning from failure |
| Dependability | "When my teammates say they'll do something, they follow through" | Clear commitments; regular check-ins; address reliability issues |
| Structure & Clarity | "Our team has an effective decision-making process" | Define roles; document processes; use OKRs |
| Meaning | "The work I do for our team is meaningful to me" | Connect work to purpose; allow autonomy; align tasks to interests |
| Impact | "I understand how our team's work contributes to the organization" | Share customer impact; celebrate wins; connect to strategy |

## Practice Exercises

### Exercise 1: Five Dysfunctions Assessment

Rate your team (1-5) on each dysfunction:

| Dysfunction | Rating | Evidence |
|-------------|--------|----------|
| Absence of Trust | | |
| Fear of Conflict | | |
| Lack of Commitment | | |
| Avoidance of Accountability | | |
| Inattention to Results | | |

Which dysfunction should you address first?

### Exercise 2: Psychological Safety Audit

Ask team members (anonymously) to rate:

1. "If I make a mistake on this team, it is held against me." (reverse scored)
2. "Members of this team are able to bring up problems and tough issues."
3. "People on this team sometimes reject others for being different." (reverse scored)
4. "It is safe to take a risk on this team."
5. "It is difficult to ask other members of this team for help." (reverse scored)
6. "No one on this team would deliberately act in a way that undermines my efforts."
7. "Working with members of this team, my unique skills and talents are valued."

Analyze results and create an action plan.

### Exercise 3: Meeting Effectiveness Review

For your last three team meetings, evaluate:

1. Did everyone participate roughly equally?
2. Were decisions made or just discussed?
3. Did anyone stay silent who should have contributed?
4. Was there productive conflict or artificial harmony?
5. What one change would improve the next meeting?
