# Lesson 4: Organizational Culture

## Module 22: Organizational Behavior | Estimated Time: 3 Hours

## Learning Objectives

By the end of this lesson, you will be able to:

1. **Define** organizational culture and its components using Schein's model
2. **Identify** different culture types and their business implications
3. **Assess** organizational culture through observable indicators
4. **Design** culture change initiatives using proven frameworks
5. **Align** culture with strategy for competitive advantage

## Introduction

Organizational culture is often described as "how we do things around here." It shapes behavior, influences decisions, and determines whether strategies succeed or fail. Culture can be a powerful competitive advantage or a significant liability - but it cannot be ignored.

## 1. Schein's Three Levels of Culture

### The Culture Iceberg

```
┌─────────────────────────────────────────────────────────────────────┐
│              SCHEIN'S LEVELS OF ORGANIZATIONAL CULTURE               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│                         ~~~~SURFACE~~~~                              │
│  ════════════════════════════════════════════════════════════       │
│                                                                       │
│  LEVEL 1: ARTIFACTS (Visible)                                        │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  Observable structures and processes:                       │    │
│  │  • Physical space and décor                                 │    │
│  │  • Dress code                                               │    │
│  │  • Published values and mission                            │    │
│  │  • Language and jargon                                      │    │
│  │  • Rituals and ceremonies                                   │    │
│  │  • Stories and legends                                      │    │
│  │  • Technology and products                                  │    │
│  │                                                              │    │
│  │  EASY TO OBSERVE, HARD TO INTERPRET                         │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ══════════════════════════════════════════════════════════════     │
│                                                                       │
│  LEVEL 2: ESPOUSED VALUES (Stated)                                   │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  Strategies, goals, philosophies (espoused justifications): │    │
│  │  • Official values statements                               │    │
│  │  • Stated priorities                                        │    │
│  │  • Norms and rules                                          │    │
│  │  • Operating philosophies                                   │    │
│  │  • Ethical codes                                            │    │
│  │                                                              │    │
│  │  MAY OR MAY NOT MATCH ACTUAL BEHAVIOR                       │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  ══════════════════════════════════════════════════════════════     │
│                                                                       │
│  LEVEL 3: BASIC UNDERLYING ASSUMPTIONS (Hidden)                      │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  Unconscious, taken-for-granted beliefs:                    │    │
│  │  • What is truth?                                           │    │
│  │  • What is human nature?                                    │    │
│  │  • How should we relate to each other?                     │    │
│  │  • What is the basis for power?                            │    │
│  │  • How should time and space be used?                      │    │
│  │                                                              │    │
│  │  INVISIBLE BUT MOST POWERFUL - SOURCE OF VALUES            │    │
│  │  AND BEHAVIOR                                               │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│                         ~~~~DEEP~~~~                                 │
│                                                                       │
│  KEY INSIGHT: To understand culture, you must go below the          │
│  surface. What people say (espoused values) often differs from      │
│  what they actually believe (basic assumptions).                     │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Decoding Culture Levels

| Level | Examples | How to Assess |
|-------|----------|---------------|
| Artifacts | Open office vs. corner offices; Casual vs. formal dress | Observation, physical walkthrough |
| Espoused Values | "Innovation is our priority"; "Customers first" | Mission statements, speeches, policies |
| Basic Assumptions | "You can only trust data"; "Hierarchy knows best" | Deep interviews, stories, observing decisions |

### Culture Type Details

| Type | Leader Style | Effectiveness Criteria | Example Companies |
|------|-------------|----------------------|-------------------|
| Clan | Facilitator, mentor | Cohesion, morale, development | Southwest Airlines, Patagonia |
| Adhocracy | Innovator, entrepreneur | Innovation, transformation, agility | Google, 3M |
| Market | Competitor, producer | Market share, profitability, beating competition | GE (Welch era), Amazon |

### Culture Diagnostic Questions

Ask: "What happens when..."

| Situation | Culture Signal |
|-----------|---------------|
| Someone makes a mistake | Blame vs. learning |
| A deadline is at risk | Collaboration vs. finger-pointing |
| A new idea is proposed | Openness vs. dismissiveness |
| Someone challenges leadership | Welcomed vs. punished |
| Resources are scarce | Hoarding vs. sharing |
| Good news arrives | Shared celebration vs. individual credit |

## 4. Culture Change

### Why Culture Change Is Hard

```

│              WHY CULTURE CHANGE IS DIFFICULT                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  CULTURE IS:                                                         │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  • INVISIBLE - Operating below consciousness                │    │
│  │  • PERVASIVE - Embedded in everything                       │    │
│  │  • STABLE - Resists change by design                       │    │
│  │  • SELF-REINFORCING - Attracts similar people              │    │
│  │  • EMOTIONAL - Connected to identity                       │    │
│  │                                                              │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
│  COMMON MISTAKES IN CULTURE CHANGE:                                  │
│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │                                                              │    │
│  │  ✗ Expecting quick results (culture change takes 3-5 years)│    │
│  │  ✗ Focusing only on values posters                         │    │
│  │  ✗ Not changing systems and structures                     │    │
│  │  ✗ Leaders not modeling new behaviors                      │    │
│  │  ✗ Underestimating emotional resistance                    │    │

│                                                                       │
│  ┌─────────────────────────────────────────────────────────────┐    │
│  │  "Culture change is behavior change at scale."              │    │
│  │                                                              │    │
│  │  Change enough individual behaviors, sustained over time,   │    │
│  │  and culture will shift.                                    │    │
│  └─────────────────────────────────────────────────────────────┘    │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Culture Change Framework

```
┌─────────────────────────────────────────────────────────────────────┐
│              CULTURE CHANGE FRAMEWORK                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                       │
│  PHASE 1: ASSESS AND DEFINE                                          │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Assess current culture (what is it really?)           │      │
│  │  • Define target culture (what does strategy require?)   │      │
│  │  • Identify gap (what must change?)                      │      │
│  │  • Prioritize critical behaviors                         │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  PHASE 2: ALIGN SYSTEMS                                              │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  Change systems to reinforce desired behaviors:           │      │
│  │  • Hiring criteria                                        │      │
│  │  • Performance metrics                                    │      │
│  │  • Promotion criteria                                     │      │
│  │  • Compensation and rewards                               │      │
│  │  • Decision-making processes                              │      │
│  │  • Physical environment                                   │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  PHASE 3: LEAD THE CHANGE                                            │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Leaders must visibly model new behaviors               │      │
│  │  • Tell new stories that embody desired culture          │      │
│  │  • Celebrate examples of new behavior                     │      │
│  │  • Address misaligned behavior immediately               │      │
│  │  • Remove persistent blockers                            │      │
│  └───────────────────────────────────────────────────────────┘      │
│                          │                                           │
│                          ▼                                           │
│  PHASE 4: SUSTAIN AND EMBED                                          │
│  ┌───────────────────────────────────────────────────────────┐      │
│  │  • Onboard new hires into new culture                     │      │
│  │  • Continue reinforcing through rituals                   │      │
│  │  • Measure culture change progress                        │      │
│  │  • Adjust approach based on feedback                      │      │
│  │  • Plan for leadership succession                         │      │
│  └───────────────────────────────────────────────────────────┘      │
│                                                                       │
│  TIMELINE: Expect 3-5 years for significant culture shift            │
│                                                                       │
└─────────────────────────────────────────────────────────────────────┘
```

### Critical Behaviors Approach

| Principle | Application |
|-----------|-------------|
| Focus on few behaviors | Choose 3-5 specific, observable behaviors |
| Make them specific | "Acknowledge input before giving opinion" vs. "Be collaborative" |
| Model from top | Leaders must demonstrate behaviors first |
| Create reinforcing mechanisms | Systems, rewards, and consequences |
| Tell stories | Share examples of behaviors leading to success |

## Practice Exercises

### Exercise 1: Culture Audit

For your organization:

1. List 5 artifacts you observe (dress code, space, rituals)
2. List 3 espoused values from official communications
3. Identify 2 underlying assumptions based on how decisions are made
4. Where do espoused values and actual behavior diverge?

### Exercise 2: Competing Values Assessment

Rate your organization (1-10) on each dimension:

| Culture Type | Score | Evidence |
|--------------|-------|----------|
| Clan (collaboration, people) | | |
| Adhocracy (innovation, flexibility) | | |
| Market (competition, results) | | |
| Hierarchy (efficiency, stability) | | |

What is your dominant culture? Does it fit your strategy?

### Exercise 3: Culture Change Design

If you needed to shift culture:

1. What specific behaviors would you change?
2. What systems would you need to modify?
3. How would leaders need to behave differently?
4. What stories would you tell?
5. What would you measure?
