# Weekly Review Template

> "Reflection is the most important part of the learning process." — John Dewey

## Purpose

The weekly review synthesizes the week's learning, assesses progress, completes the application exercise, and prepares for the week ahead. This is where daily practices become lasting change.

Time required: 30-45 minutes (Day 7 of each week)

## Review Best Practices

### Environment
- Same time each week (protect this appointment)
- Quiet, uninterrupted space
- All materials ready (journal, week file, reading)

### Mindset
- Approach with curiosity, not judgment
- Celebrate progress, learn from setbacks
- Be specific, not vague
- Be honest—this is for you

### Follow-Through
- If reading is incomplete, schedule catch-up time
- If scores are consistently low, adjust your morning practice
- If insights aren't translating to action, focus on smaller applications
- If the process feels stale, revisit mentor profiles for inspiration
