## How to Use This Curriculum

### Recommended Approach

1. **Start with Phase 0** - The consulting frameworks provide the analytical toolkit you'll use throughout
2. **Progress sequentially** - Phases build on each other
3. **Complete all components** - Topics → Practice Problems → Cases → Assessment
4. **Apply immediately** - Use frameworks in your actual work
5. **Track progress** - Use PROGRESS.md to monitor completion

### For Each Module

```
1. Read the Module README (learning objectives, overview)
2. Work through each topic in order
3. Complete practice problems for each topic
4. Analyze the case studies
5. Take the module assessment
6. Reflect on CEO applications
```

### Time Investment

| Activity | Time per Module |
|----------|----------------|
| Topic reading | 3-4 hours |
| Practice problems | 2-3 hours |
| Case analysis | 2-3 hours |
| Assessment | 1 hour |
| **Total** | **8-11 hours** |
