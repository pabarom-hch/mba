# Module 17: Brand Management

## Overview

Brand management is the art and science of creating, developing, and maintaining valuable brands. Strong brands command premium prices, attract loyal customers, and create sustainable competitive advantage. This module covers brand strategy, equity measurement, architecture, and extension—providing frameworks to build and protect brand value.

## Learning Objectives

By the end of this module, you will be able to:

1. Define and measure brand equity
2. Develop brand positioning and identity
3. Design brand architecture for portfolios
4. Evaluate and execute brand extensions
5. Manage brands across touchpoints
6. Build and maintain customer-brand relationships
7. Value brands and protect brand equity

```
BRAND EQUITY MODEL (KELLER):
─────────────────────────────────────────────────────
                    BRAND
                   EQUITY
                      │
        ┌─────────────┴─────────────┐
        │                           │
   BRAND AWARENESS           BRAND IMAGE
        │                           │
   ┌────┴────┐              ┌───────┴───────┐
   │         │              │               │
Recognition Recall    Associations   Attitudes
   │         │              │               │
   └─────────┘              └───────────────┘

BRAND AWARENESS:
─────────────────────────────────────────────────────
• Recognition: Identify brand when exposed
• Recall: Retrieve brand from memory
• Top-of-mind: First brand that comes to mind

BRAND IMAGE:
─────────────────────────────────────────────────────
• Associations: What the brand means
• Attributes: Product-related and non-product
• Benefits: Functional, experiential, symbolic
• Attitudes: Overall evaluation
```

## Brand Value Creation

```
HOW BRANDS CREATE VALUE:
─────────────────────────────────────────────────────

FOR CUSTOMERS:
─────────────────────────────────────────────────────
• Reduce risk (quality assurance)
• Simplify choice (decision shortcut)
• Express identity (self-expression)
• Create meaning (emotional connection)
• Enable belonging (community)

FOR COMPANIES:
─────────────────────────────────────────────────────
• Price premium (higher margins)
• Customer loyalty (repeat purchase)
• Trade leverage (retailer power)
• Extension platform (new products)
• Talent attraction (employer brand)
• Resilience (crisis protection)

BRAND VALUE EQUATION:
─────────────────────────────────────────────────────
Brand Value =
  (Price Premium × Volume) +
  (Loyalty Effect × Customer Lifetime) +
  (Extension Value) +
  (Reduced Risk Premium)
```
