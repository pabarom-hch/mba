# Learning and Memory

## Learning Objectives

- Understand how consumers learn about brands
- Apply conditioning principles to marketing
- Design memorable marketing communications
- Build and protect brand associations in memory
- Use retrieval cues effectively

---

## Introduction

Consumer learning is the process by which individuals acquire knowledge and experience that shapes future purchasing behavior. Memory stores this learning for later retrieval. Understanding how consumers learn and remember is essential for building strong brands and influencing long-term purchase behavior.

---

## Learning Theories

### Classical Conditioning

```
CLASSICAL CONDITIONING:
─────────────────────────────────────────────────────

PAVLOV'S MODEL:
─────────────────────────────────────────────────────
Unconditioned Stimulus (US) → Unconditioned Response (UR)
(Food)                        (Salivation)

US + Neutral Stimulus (NS)
(Food) + (Bell)

After repeated pairing:
Conditioned Stimulus (CS) → Conditioned Response (CR)
(Bell alone)                (Salivation)

MARKETING APPLICATION:
─────────────────────────────────────────────────────
US (Pleasant music) → UR (Positive feeling)

US + NS (Brand shown with music)

CS (Brand alone) → CR (Positive feeling)

EXAMPLES:
─────────────────────────────────────────────────────
• Celebrity + Brand → Positive feelings transfer
• Music + Commercial → Emotional association
• Attractive imagery + Product → Appeal transfers

CONDITIONING REQUIREMENTS:
─────────────────────────────────────────────────────
• Repetition (multiple pairings)
• Contiguity (close in time/space)
• Forward conditioning (US before NS)
• Stimulus generalization (similar stimuli work)
```

### Operant Conditioning

```
OPERANT CONDITIONING:
─────────────────────────────────────────────────────

SKINNER'S MODEL:
─────────────────────────────────────────────────────
Behavior → Consequence → Future behavior affected

REINFORCEMENT TYPES:
─────────────────────────────────────────────────────
                    Add             Remove
─────────────────────────────────────────────────────
Increase behavior   Positive        Negative
                    Reinforcement   Reinforcement
                    (Rewards)       (Relief)

Decrease behavior   Positive        Negative
                    Punishment      Punishment
                    (Penalties)     (Remove good)
─────────────────────────────────────────────────────

MARKETING APPLICATIONS:
─────────────────────────────────────────────────────
Positive reinforcement:
• Loyalty rewards
• Free samples
• Cash-back offers
• Points programs
• "Thank you" messaging

Negative reinforcement:
• Problem relief (headache medicine)
• Anxiety reduction (insurance)
• Status recovery (premium products)

Shaping (successive approximation):
• Sign up → Get reward
• Make first purchase → Get discount
• Become regular → Get loyalty status
• Become advocate → Get VIP treatment

REINFORCEMENT SCHEDULES:
─────────────────────────────────────────────────────
Fixed ratio:      Every X purchases gets reward
Variable ratio:   Random chance of reward (gambling)
Fixed interval:   Monthly statement rewards
Variable interval: Surprise rewards (most engaging)
```

## Memory

### Memory Systems

```
MEMORY MODEL:
─────────────────────────────────────────────────────

                    ┌─────────────┐
     Sensory ──────►│   Short-    │
     Input         │    Term     │
                   │   Memory    │
                   │  (Working)  │
                   └──────┬──────┘
                          │
                   Encoding│Rehearsal
                          │
                   ┌──────▼──────┐
                   │   Long-     │
                   │    Term     │◄─── Retrieval
                   │   Memory    │
                   └─────────────┘

SENSORY MEMORY:
─────────────────────────────────────────────────────
• Very brief (milliseconds to seconds)
• High capacity but rapid decay
• Iconic (visual) and echoic (auditory)
• Preconscious processing

SHORT-TERM MEMORY (Working):
─────────────────────────────────────────────────────
• Limited capacity (7±2 items)
• Limited duration (15-30 seconds)
• Active processing
• Chunking increases capacity

Marketing implication:
Phone numbers, product codes should be <7 digits
Chunk information meaningfully

LONG-TERM MEMORY:
─────────────────────────────────────────────────────
• Essentially unlimited capacity
• Permanent storage (but access varies)
• Requires encoding and organization
• Episodic (events) and semantic (facts)
```

### Memory Encoding

```
ENCODING STRATEGIES:
─────────────────────────────────────────────────────

ELABORATION:
─────────────────────────────────────────────────────
Deeper processing → Stronger memory
• Connect to existing knowledge
• Create meaning
• Self-reference (personal relevance)

Levels of processing:
Shallow → Medium → Deep
(Font style) (Rhyming) (Meaning)

Marketing use:
• Interactive content
• Self-reference ("Imagine yourself...")
• Storytelling

DISTINCTIVENESS:
─────────────────────────────────────────────────────
Unique stimuli are better remembered
• Von Restorff effect (isolation effect)
• Unusual ads stand out
• Breaking category norms

Marketing use:
• Unique brand assets
• Unexpected creative
• Breaking the mold

EMOTIONAL AROUSAL:
─────────────────────────────────────────────────────
Emotional experiences are better remembered
• Flashbulb memories
• Mood-congruent encoding
• Both positive and negative work

Marketing use:
• Emotional advertising
• Experiential marketing
• Memorable brand experiences

IMAGERY:
─────────────────────────────────────────────────────
Images are better remembered than words
• Dual coding (visual + verbal)
• Concrete > Abstract
• Vivid imagery creates strong traces

Marketing use:
• Visual branding
• Memorable imagery in ads
• Product visualization
```

### Memory Retrieval

```
RETRIEVAL PROCESSES:
─────────────────────────────────────────────────────

RECOGNITION VS. RECALL:
─────────────────────────────────────────────────────
Recognition:  Easier (Is this familiar?)
Recall:       Harder (What can you remember?)

Marketing implication:
Brand recognition ≠ Brand recall
Point of purchase: Recognition works
Spontaneous choice: Recall needed

RETRIEVAL CUES:
─────────────────────────────────────────────────────
Cues that trigger memory access:
• Environmental cues (store, context)
• Semantic cues (category, occasion)
• Sensory cues (smell, music, color)
• Brand cues (logo, tagline, jingle)

State-dependent retrieval:
Memory better when encoding and retrieval
contexts match

Marketing use:
• Consistent brand assets
• Contextual advertising
• Occasion-based marketing
• Packaging as memory cue

SPREADING ACTIVATION:
─────────────────────────────────────────────────────
Activating one concept activates related concepts

            ┌──── Refreshing ────┐
            │                    │
         Coca-Cola ──── Red ────┘
            │          │
            │          └──── Happiness
            │
            └──── Soda ──── Sweet
```

```
BRAND MEMORY BUILDING:

DISTINCTIVENESS:
─────────────────────────────────────────────────────
Create unique, ownable brand assets:

• Color palette
• Sound/music
• Characters/mascots

─────────────────────────────────────────────────────
Mere exposure effect + Memory consolidation

Enough to build memory, not so much to annoy

─────────────────────────────────────────────────────

CONTEXT:
─────────────────────────────────────────────────────
Build associations with usage situations

RECENCY:
─────────────────────────────────────────────────────
Recent exposure increases salience
• Always-on marketing
• Retargeting
```

## Forgetting and Interference

### Why We Forget

```
FORGETTING THEORIES:
─────────────────────────────────────────────────────

DECAY:
─────────────────────────────────────────────────────
Memory traces fade over time without use
Solution: Repetition, reinforcement

INTERFERENCE:
─────────────────────────────────────────────────────
Other memories block retrieval

Proactive interference:
Old learning blocks new learning
(Old brand habits interfere with new brand trial)

Retroactive interference:
New learning blocks old learning
(Competitor advertising interferes with your brand)

RETRIEVAL FAILURE:
─────────────────────────────────────────────────────
Information is there but can't be accessed
• Missing cues
• Tip-of-the-tongue
• Context mismatch

Solution: Provide retrieval cues

FORGETTING CURVE:
─────────────────────────────────────────────────────
Ebbinghaus curve shows rapid initial forgetting

Memory
   │
100│*
   │ *
 50│  *
   │   *****
 20│        **********
   │                   **************
   └─────────────────────────────────→ Time
      1hr  1day  1week  1month

Marketing implication:
• Space repetition over time
• Multiple exposures needed
• Refresh before forgetting
```

### Competitive Interference

```
MANAGING COMPETITIVE INTERFERENCE:
─────────────────────────────────────────────────────

PROBLEM:
─────────────────────────────────────────────────────
Competitor advertising can:
• Create confusion
• Block your brand memory
• Steal associations

SOLUTIONS:
─────────────────────────────────────────────────────

1. DISTINCTIVE BRAND ASSETS:
   Make your brand unmistakably unique
   Can't be confused with competitors

2. CONSISTENT EXECUTION:
   Same assets over time build stronger memory
   Less susceptible to interference

3. CATEGORY ENTRY POINTS:
   Link brand to specific usage situations
   Own the occasion in memory

4. RECENCY ADVANTAGE:
   More recent exposure = More accessible
   Maintain consistent presence

5. DEPTH OF PROCESSING:
   Elaborated memories are more resistant
   Engage customers meaningfully
```

## Key Takeaways

1. **Condition positive associations** - Pair brand with positive stimuli
2. **Reinforce desired behaviors** - Reward customers for action
3. **Encode deeply** - Create meaningful, emotional processing
4. **Build distinctive assets** - Own unique memory cues
5. **Maintain presence** - Fight forgetting with consistency
6. **Provide retrieval cues** - Make your brand easy to remember
