## Finance Frameworks

### DuPont Analysis
```
ROE = Net Margin × Asset Turnover × Equity Multiplier

ROE = (Net Income/Revenue) × (Revenue/Assets) × (Assets/Equity)
```
**Use for**: Return on equity decomposition

### DCF Valuation
```
              CF₁        CF₂        CFₙ + TV
Value = ────────── + ────────── + ──────────
         (1+r)¹      (1+r)²      (1+r)ⁿ

Terminal Value = CFₙ × (1+g) / (r-g)
```
**Use for**: Intrinsic valuation

### WACC Formula
```
WACC = (E/V × Re) + (D/V × Rd × (1-T))

E = Equity, D = Debt, V = E+D
Re = Cost of equity, Rd = Cost of debt
T = Tax rate
```
**Use for**: Discount rate calculation

### Capital Budgeting Rules
```
NPV > 0           → Accept
IRR > WACC        → Accept
Payback < Target  → Accept
PI > 1            → Accept
```
**Use for**: Investment decisions

## Operations Frameworks

### Little's Law
```
L = λ × W

L = Inventory/WIP
λ = Throughput rate
W = Flow time
```
**Use for**: Process analysis

### EOQ Formula
```
EOQ = √(2DS/H)

D = Annual demand
S = Order cost
H = Holding cost per unit
```
**Use for**: Inventory optimization

### OEE (Overall Equipment Effectiveness)
```
OEE = Availability × Performance × Quality
```
**Use for**: Equipment efficiency

### DMAIC (Six Sigma)
```
Define → Measure → Analyze → Improve → Control
```
**Use for**: Process improvement

### Seven Wastes (Lean)
```
T - Transportation
I - Inventory
M - Motion
W - Waiting
O - Overproduction
O - Overprocessing
D - Defects
```
**Use for**: Waste identification

## Quick Decision Formulas

### Breakeven Analysis
```
Breakeven Units = Fixed Costs / (Price - Variable Cost)
Breakeven Revenue = Fixed Costs / Contribution Margin %
```

### Payback Period
```
Payback = Initial Investment / Annual Cash Flow
```

### Market Share
```
Market Share = Company Sales / Total Market Sales
```

### Contribution Margin
```
CM = Price - Variable Cost
CM% = (Price - VC) / Price
```

### Gross Margin
```
Gross Margin = (Revenue - COGS) / Revenue
```

### Customer Acquisition Cost
```
CAC = Marketing Spend / New Customers Acquired
```

### Return on Investment
```
ROI = (Gain - Cost) / Cost × 100%
```
